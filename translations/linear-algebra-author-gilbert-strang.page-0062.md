## 2.3. 使用矩陣進行消去法 59

「交換律」3 x 4 = 4 x 3 看起來很明顯。但 *EA* 通常與 *AE* 不同，因為 *E* 作用於 *A* 的欄上。

結合律成立  *A(BC) = (AB)C*
交換律不成立  *Often AB ≠ BA*

還有另一項矩陣乘法的要求。假設 *B* 只有一個欄（這個欄是 *b*）。矩陣-矩陣法則要求 *EB* 應該與矩陣-向量法則對於 *Eb* 一致。即使如此，我們應該能夠一次將矩陣 *EB* 乘以一個欄：

如果 *B* 具有欄 *b₁*, *b₂*, *b₃*，那麼 *EB* 的欄是 *Eb₁*, *Eb₂*, *Eb₃*。

矩陣乘法  *AB = A[b₁ b₂ b₃] = [Ab₁ Ab₂ Ab₃]*. (3)

這對於矩陣乘法的公式 (2) 成立。如果你將 *A* 的第 3 欄乘以 *E*，你將得到 *EA* 的第 3 欄：

```
[ 1  0  0 ]   [ -2 ]   [ -2 ]
[ -2 1  0 ] * [ -3 ] = [  1 ]
[ 0 0  1 ]   [  7 ]   [  7 ]
```

*E*(欄 *j* of *A*) = 欄 *j* of *EA*.

這個要求處理的是欄，而這與公式 (2) 中的欄位相乘一致。下一節將描述 *AB* 的每一項的乘積 *A* *B*。矩陣乘法的重點是所有方法（列、欄、整個矩陣）都適用。

### 矩陣 *P<sub>ij</sub>* 用於列交換

為了交換 *i* 列和 *j* 列，我們使用 *E<sub>ij</sub>*。要交換或“排列”這些列，我們需要一個零在主對角線上，而主對角線上的列包含一個非零元素。通過交換這兩列，我們有主元和消去法向前發展。

什麼是 *P<sub>23</sub>* 交換第 2 列和第 3 列？我們可以通過交換單位矩陣 *I* 的列來找到它：

排列矩陣

```
P₂₃ = [ 1  0  0 ]
      [ 0  0  1 ]
      [ 0  1  0 ]
```

這是一個列交換矩陣。將 *P<sub>23</sub>* 乘以一個列向量會交換第 2 和第 3 個分量：

```
[ 1  0  0 ]   [ 1 ]   [ 1  2  4 ]
[ 0  0  1 ] * [ 5 ] = [ 5  0  6 ]
[ 0  1  0 ]   [ 3 ]   [ 3  1  5 ]
```

如你所見，*P<sub>23</sub>* 做了它所要做的。與零在主對角線上的 *E* 不同，*P<sub>23</sub>* 是可逆的，並且它的逆矩陣是它自己。
