**問題 11-20 研究 3x3 系統的消去法（以及可能的失敗情形）。**

**11 (推薦)** 一個線性方程組系統恰好有兩個解。為什麼？
(a) 如果 (x, y, z) 和 (X, Y, Z) 是兩個解，那麼另一個解是什麼？
(b) 如果 25 個平面在兩點相交，這些點在哪裡相遇？

**12** 透過兩個列運算將此系統化為上三角形式：
```
2x + 3y + z = 8
4x + 7y + 5z = 20
-2y + 2z = 0
```

圈出主元。使用回代法解出 z, y, x。

**13** 應用消去法（圈出主元）和回代法解出：
```
2x - 3y = 3
4x - 5y + z = 7
2x - y - 3z = 5
```

列出三個列運算：減去 ____ 次 ____ 從 ____ 列。

**14** 哪個數字 *d* 迫使進行列交換，以及什麼是三角系統（非奇異）的三角形式？哪個 *d* 使此系統變成奇異的（沒有第三個主元）？
```
2x + 5y + z = 0
4x + dy + z = 2
y - z = 3
```

**15** 哪個數字 *b* 導致列交換？哪個 *b* 導致缺少主元？
在這種奇異情況下找到一個非零解 x, y, z。
```
x + by = 0
x - 2y - z = 0
y + z = 0
```

**16** (a) 建構一個 3x3 系統，需要兩個列交換才能達到三角形式，並找到一個解。
(b) 建構一個 3x3 系統，需要一個列交換來繼續進行，但之後會崩潰。

**17** 如果列 1 和列 2 是相同的，如何透過列運算消除（允許列交換）？如果列 1 和列 2 是相同的，如何避免消去法出錯？
