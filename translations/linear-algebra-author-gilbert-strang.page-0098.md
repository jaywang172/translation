**2.6 消去法：分解為 A = LU**

學生們常常覺得數學課程過於理論化。但並非如此，本節將會是極具實際意義的。高斯消去法是線性代數中最有用的方法之一，許多關鍵的線性代數問題，當我們仔細觀察時，其實都是矩陣分解。原始矩陣 A 可以分解成兩個或三個特殊矩陣的乘積。最常被使用的分解——也是在實務上最重要的分解——來自消去法，即三角矩陣 L 和 U 的分解。從消去法得到的分解稱為 **A = LU**。

我們已經知道 U 是上三角矩陣，其對角線上的元素為消去法的中心點。消去法的步驟是將 A 分解為 L 和 U。我們將展示如何透過反轉消去步驟（將 U 變回 A）來得到下三角矩陣 L。L 的元素是消去法中用來乘以中心點，並從其下方元素中減去的乘數 *l<sub>ij</sub>*。

從一個以 2 為例子的矩陣 A = [2 1 6; -3 1 0; 5 6 8] 開始。我們需要消去第一行中的 2。將第一行乘以 2，然後從 U 的第二行和第三行中減去。

前向從 A 到 U：
E<sub>21</sub>A = [1 0 2; -3 1 6; 5 6 8] = [2 1 6; 0 5 0; 0 0 8] = U

反向從 U 到 A：
E<sub>21</sub><sup>-1</sup>U = [1 0 0; 3 1 0; 0 0 1] [2 1 6; 0 5 0; 0 0 8] = [2 1 6; -3 1 0; 5 6 8] = A。

第二個線索來自我們的分解 *LU = A*。為了得到 E<sub>21</sub>，我們寫下 A。移動到更大的矩陣時，會有更多的 E。然後 L 將會包含所有 E 的反矩陣。

每個步驟從 A 到 U 都乘以一個矩陣 E<sub>ij</sub>，以產生第 (i, j) 位置的零。為了保持清晰，我們將以最常見的情況為例——不需交換行。E<sub>21</sub> 和 E<sub>32</sub> 的乘數 *l<sub>ij</sub>* 產生第二行和第三行（以及第一行）中的零——這就是消去法的過程。消去法結束時，U 將會是上三角矩陣。

現在讓我們來看看 E 的反矩陣。它們的反矩陣乘以 U：

E<sub>32</sub> E<sub>21</sub> A = LU。

如果消去法中沒有交換行，那麼 L 就是下三角矩陣，其對角線上的元素都是 1。L 的元素 *l<sub>ij</sub>* 是 E<sub>ij</sub> 的乘數。因此，L = [1 0 0; l<sub>21</sub> 1 0; l<sub>31</sub> l<sub>32</sub> 1]。

在我們的例子中，E<sub>21</sub> = [1 0 0; 3 1 0; 0 0 1]，因此 E<sub>21</sub><sup>-1</sup> = [1 0 0; -3 1 0; 0 0 1]。
E<sub>32</sub> = [1 0 0; 0 1 0; 0 l<sub>32</sub> 1]，因此 E<sub>32</sub><sup>-1</sup> = [1 0 0; 0 1 0; 0 -l<sub>32</sub> 1]。
因此，L = E<sub>21</sub><sup>-1</sup> E<sub>32</sub><sup>-1</sup> = [1 0 0; -3 1 0; 0 0 1] [1 0 0; 0 1 0; 0 -6 1] = [1 0 0; -3 1 0; 0 -6 1]。
