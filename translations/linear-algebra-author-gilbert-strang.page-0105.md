## 102 章：解線性方程組

您可能注意到 MATLAB 命令 `lu(pascal(4))` 產生的 L 和 U 並非總是您期望的。該子例程會選擇給定列中最大的可用樞軸元素。第二個樞軸元素可能會從 1 變為 3。但「Cholesky 分解」不會有列交換：`chol(pascal(4))`。

P = LU 的完整證明可在網頁 web.mit.edu/18.06 上找到，該網頁也提供 MIT 的 OpenCourseWare at ocm.mit.edu。這些 Pascal 矩陣具有許多非凡的特性——我們將在本書中看到。

**2.6 B** 問題是：求解 Px = b = (1, 0, 0, 0)。這右側是 I 的第一列。這意味著 x 將是 P⁻¹ 的第一列。這是高斯-喬丹消去法匹配 P 的列，即 PP⁻¹ = I。我們已經知道 Pascal 矩陣可以分解成 L 和 U 的乘積。

**兩個三角系統**

Lc = b (前向)
Ux = c (反向)

**解**

下三角系統 Lc = b 透過從上到下求解得到：

c₁ = 1
c₁ + c₂ = 0  給予  c₂ = -1
c₁ + 2c₂ + c₃ = 0  給予  c₃ = +1
c₁ + 3c₂ + 3c₃ + c₄ = 0  給予  c₄ = -1

前向消去法是矩陣乘法 L⁻¹b。它產生上三角系統 Ux = c。解 x 總是透過反向代入得到：

x₁ + x₂ + x₃ + x₄ = 1
x₂ + 2x₃ + 3x₄ = -1  給予  x₁ = +4
x₃ + 3x₄ = -1  給予  x₂ = -6
x₄ = -1  給予  x₃ = +4
x₄ = -1
