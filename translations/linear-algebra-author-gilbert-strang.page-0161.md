**第 3 章：向量空間與子空間**

**完全解**

另一個極端情況是滿秩。此時 $Ax = b$ 只有一個解或無限多個解。在這種情況下，矩陣是短而寬的 ($m \ge n$)。一個矩陣若 $r = m$（“線性獨立”）則具有滿秩。每一列都有一個樞紐元，這是一個例子。

**例 2** 有 $n = 3$ 個未知數，但只有 $m = 2$ 個方程式：

```
x + y + z = 3
x + 2y = 4
```

**滿秩**  **滿秩** ($r = m = 2$)

這些是在 $xyz$ 空間中的兩個平面。這兩個平面不是平行的，因此它們會相交成一條線。這條線上的解是消除法無法找到的解。特定解是這條線上的一個點。加上零空間向量 $x_n$ 會沿著這條線移動。因此 $x = x_p + x_n$ 給出了所有解。

我們找到 $x_p$ 和 $x_n$，透過消除 $[A \ b]$。從第 1 列減去第 2 列，並從第 2 列減去第 1 列：

```
[ 1  1  3 ]  -> [ 1  1  3 ]  -> [ 1  0  3 ] = [ R  d ]
[ 1  2  4 ]  -> [ 0 -2  1 ]  -> [ 0 -2  1 ]
```

特定解將自由變數設為 0。因此，特定解是 $x_3 = 1$。

$x_{special}$ 來自第三列（自由列）的 $R$: $s = (-3, 2, 1)$

驗證 $x_p$ 和 $x_s$ 滿足原始方程式 $Ax = b$ 和 $As = 0$:

```
2 + 1 = 3    -3 + 2 + 1 = 0
2 + 2 = 4    -3 + 4 - 1 = 0
```

零空間解 $x_n$ 是 $s$ 的倍數。它沿著解的線移動，從特定解 $x_p$ 開始。

**請注意再次檢查以寫下答案：**

**完全解**

```
x = x_p + x_n = [ 2 ] + x_3 [ -3 ]
                [ 1 ]         [ 2 ]
                [ 0 ]         [ 1 ]
```

這條線在圖 3.3 中繪製。任何點都在這條線上，我們選擇特定解 $x_3 = 0$。

特定解是固定的（不乘以常數）。特定解是唯一的，並且是常數向量。

總結一下，如果矩陣具有滿秩，則 $Ax = b$ 是確定性的（沒有自由變數）。如果 $m < n$，則方程式 $Ax = b$ 是欠定的。
