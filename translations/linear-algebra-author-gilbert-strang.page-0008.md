### 1.1. 向量與線性組合

將向量寫成列形式（在括號中）是為了節省空間。但 $v = (1, 1, -1)$ 並不是一個列向量！它實際上是一個行向量，只是暫時地橫躺著。列向量 $[1 \ 1 \ -1]$ 截然不同，儘管它具有相同的三個分量。那個列向量是行向量 $v$ 的「轉置」。

在三維空間中，$v + w$ 仍然是逐分量地計算。其和的分量為 $v_1 + w_1$、 $v_2 + w_2$ 和 $v_3 + w_3$。你會知道如何在四維、五維或 $n$ 維空間中相加向量。當 $w$ 從 $v$ 的末端開始時，第三邊是 $v + w$。沿著平行四邊形的另一條路徑是 $w + v$。問題：這四條邊都位於同一個平面上嗎？是的。而和 $v - w$ 完全繞一圈來產生 零 向量。三個向量在三維空間中的典型線性組合是 $u + 4v - 2w$。

### 重要的問題

對於一個向量 $u$，唯一的線性組合是其倍數 $cu$。對於兩個向量，組合是 $cu + dv$。對於三個向量，組合是 $cu + dv + ew$。你會從一個組合跨出大步到所有組合嗎？每個 $c$、 $d$ 和 $e$ 都是允許的。假設向量 $u, v, w$ 都在三維空間中：

1.  所有組合 $cu$ 的圖像是什麼？
2.  所有組合 $cu + dv$ 的圖像是什麼？
3.  所有組合 $cu + dv + ew$ 的圖像是什麼？

答案取決於特定的向量 $u, v$ 和 $w$。如果它們是零向量（一個非常極端的情況），那麼每個組合都將是零。如果它們是典型的非零向量（分量是隨機選擇的），那麼以下是三個答案。這是我們學科的關鍵：

1.  組合 $cu$ 填充一條線。
2.  組合 $cu + dv$ 填充一個平面。
3.  組合 $cu + dv + ew$ 填充三維空間。

零向量 $(0,0,0)$ 在線上，因為 $c$ 可以是零。它在平面上，因為 $c$ 和 $d$ 都可以是零。向量 $cu$ 的線是無限長的（向前和向後）。我特別請你思考的是所有 $cu + dv$ 的平面（在三維空間中組合兩個向量）。

將一條線上的所有 $cu$ 與另一條線上的所有 $dv$ 相加，就填充了圖 1.3 中的平面。

當我們包含第三個向量 $w$ 時，倍數 $ew$ 會產生第三條線。假設第三條線不在 $u$ 和 $v$ 的平面中。那麼將所有 $ew$ 與所有 $cu + dv$ 組合起來，就會填充整個三維空間。
