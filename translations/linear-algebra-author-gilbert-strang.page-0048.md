## 2.2. 消去法的概念

2.2 消去法的概念

本章說明了一種解聯立方程式的方法。這種方法稱為「消去法」，你可以在我們的例子 2 中立即看到它。在消去法之前，x 和 y 出現在兩個方程式中。經過消去後，第一個未知數 x 從第二個方程式 8y = 8 中消失了。

**消去前**
```
x - 2y = 1
3x + 2y = 11
```

**消去後**
```
x - 2y = 1   (將方程式 1 乘以 3)
8y = 8       (從方程式 1 減去 3x)
```

新的方程式 8y = 8 立即給出 y = 1。將此值代入第一個方程式，得到 x - 2 = 1。因此 x = 3，解為 (x, y) = (3, 1) 完整。

消去法產生一個**三角形系統**——這是我們的目標。非零係數 1, 2, 8 形成一個三角形。該系統是從底部向上解的——首先 y = 1，然後 x = 3。此快速過程稱為**回代**。任何三角形系統的上半部分，在消去後都呈現三角形。

重要提示：原始方程式具有相同的解 x = 3 和 y = 1。圖 2.5 顯示每個系統作為一對直線，在解點 (3, 1) 處相交。經過消去後，這些線仍然在同一個點相交。每一步都經過正確的方程式運算。

**我們如何從第一個方程式得到第二個方程式？** 我們從第一個方程式減去 3 倍的方程式 2。這個步驟是消去方程式 2 中的 x 的基本運算。我們經常使用它，因此我們仔細地觀察它：

**消去 x：** 從方程式 1 減去方程式 2 的倍數。

三倍方程式 x - 2y = 1 得到 3x - 6y = 3。當這個方程式從 3x + 2y = 11 中減去時，左側變成 -8y。主要點是 3x 取消了 3x。什麼仍然存在？右側是 2y - (-6y) 或 8y，而 x 被消去。這個系統變成了三角形。

**問一下自己：** 哪個乘數 *l* = 3 被找到？第一個方程式包含 1x。因此，第一個方程式乘以 3。第二個方程式包含 3x，所以乘數為 3。然後減去 3x 產生了零和三角形。

**表格 2.1：** 消去過程

| **消去前** | **消去後** |
|---|---|
| ``` x - 2y = 1  3x + 2y = 11 ``` | ``` x - 2y = 1  8y = 8 ``` |
|  | y = 1 |
|  | x = 3 |

**圖 2.5：** 消去法。兩條線 x - 2y = 1 和 3x + 2y = 11 相交於 (3, 1)。
