**第 3 章：向量空間與子空間**

[圖片 3.1：「四維」矩陣空間 M。 “零維”空間 Z。]

圖中顯示一個矩陣空間 M，其中一個典型的向量表示為 \[ \begin{bmatrix} a \\ b \\ c \\ d \end{bmatrix} \]，以及一個最小的向量，即零向量。

在三維空間中，一個平面並非 R²（即使它看起來像 R²）。向量具有三個分量，並且屬於 R³。這個平面是一個向量空間 R³ 的子空間。

這說明了線性代數中最基本的概念之一：通過 (0, 0, 0) 的平面是整個向量空間 R³ 的一個子空間。

**定義：** 一個向量空間的子空間是一個向量集合（包含 **0** 向量），滿足以下兩個要求：

(i) 如果 **v** 和 **w** 是子空間中的向量，那麼 **v** + **w** 也在子空間中。

(ii) 如果 **v** 是子空間中的向量，而 *c* 是一個純量，那麼 *c***v** 也在子空間中。

換句話說，這個向量集合在加法 **v** + **w** 和乘法 *c***v** 下是“封閉”的。這些運算使向量保持在子空間中。我們也可以進行減法，因為 -**w** 也在子空間中，並且它的和 **v** + (-**w**) = **v** - **w** 也在子空間中。簡而言之，所有線性組合都保持在子空間中。

所有這些運算都遵循主空間的規則，因此所需的八個條件是自動的。我們只需要檢查子空間的要求，以便我們可以進行線性組合。

**第一條規則：每個子空間都包含零向量。** R³ 中的平面必須通過 (0, 0, 0)。我們特別強調這一點，因為它直接從規則 (ii) 推導出來。選擇 *c* = 0，則要求 0**v** 也在子空間中。

不包含原點的平面將無法通過測試。當 **v** 是這樣一個平面上的向量，而 -**v** 不在平面上時。

通過原點的直線也是子空間。當我們乘以 5，或添加任何向量到該直線上時，我們仍然停留在該直線上。但該直線必須通過 (0, 0, 0)。
