**12** 假設 *AB* = *BA* 且 *AC* = *CA* 對於以下兩個特定矩陣 *B* 和 *C* 成立：

       [ 0 0 ]

       [ 0 0 ]

**13** 下列哪些矩陣式子保證相等？( *A* - *B* )²; *A*² - *B*²; ( *B* - *A* )²; *A*² - 2*AB* + *B*²; *A*( *A* - *B* ) - *B*( *A* - *B* ); *A*² - *AB* - *BA* + *B*²?

**14** 真或假：

(c) 如果 *AB* 和 *BA* 有定義，則 *A* 和 *B* 都是方陣。
(d) 如果 *AB* = *B*，則 *A* = *I*。
