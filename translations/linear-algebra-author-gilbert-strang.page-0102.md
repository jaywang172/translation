## 2.6. 消去法 = 分解因式：A = LU 99

第一次消去法的步驟是將第一欄的元素歸零，產生主軸元素下方的零。為了找到每個新項的主軸元素，我們需要一次乘法和一次減法。我們將計算這個第一步驟所需的 $n^2$ 次乘法和 $n^2$ 次減法。實際上，這會少 $n$，因為第一列不會改變。

下一個步驟清除第二欄的主軸元素。工作矩陣的大小現在是 $n-1$。估計這個步驟需要 $(n-1)^2$ 次乘法和減法。隨著消去法向前進行，矩陣會變得越來越小。粗略計算到達 $n$ 的平方和需要 $1^2 + 2^2 + ... + n^2$。

存在一個公式來計算 $1^2 + 2^2 + ... + n^2 = \frac{n(n+1)(2n+1)}{6} = \frac{1}{3}n^3 + \frac{1}{2}n^2 + \frac{1}{6}n$。對於平方和，$\frac{1}{3}$ 項最重要，而 $\frac{1}{2}$ 和 $\frac{1}{6}$ 不重要。平方和的總和就像 $x^2$ 的積分從 0 到 $n$：$\int_0^n x^2 dx = \frac{1}{3}n^3$。

**消去法 A 需要約 $\frac{1}{3}n^3$ 次乘法和 $\frac{1}{3}n^3$ 次減法。**

那麼右側的 b 呢？向前進行，我們從較低分量 $b_2, ..., b_n$ 中減去較高分量。這個第二步驟只需要 $n-1$ 步驟。第二步驟只需要 $n-1$ 步驟，因為 $b_1$ 沒有參與。最後一步是向前消去，需要一個步驟。

現在開始回代。計算 $x_n$ 需要一個步驟（除以主軸元素）。下一個未知數需要兩個步驟。當我們到達 $x_1$ 時，它需要 $n$ 步驟（$n-1$ 次替換）。從 $b$ 到 $c$ 到 $x$——向前到底部，再回到頂部——確切地需要 $n^2$。

$[ (n-1) + (n-2) + ... + 1 ] + [ 1 + 2 + ... + (n-1) + n ] = n^2$. (6)

為了總結這個總和，將 $(n-1)$ 與 $1$ 和 $(n-2)$ 與 $2$ 配對。配對留下 $n$ 項，每個項等於 $n$。這使得 $n^2$。右側的成本比左側少 $n$。

**Solve**  每個右側需要 $n$ 次乘法和 $n$ 次減法。

一個帶狀矩陣 B 只有在非零對角線下方和主對角線上的元素。消去法只會影響主對角線。消去法的第一個步驟需要 $2n$ 次乘法和減法（在零上方和下方）。清除下一欄需要 $n-1$ 次乘法和減法。總之，消去法需要 $3n$ 次乘法和減法。

**Solve**  需要約 $\frac{1}{2}n^2$ 次乘法和 $\frac{1}{2}n^2$ 次減法。

**高斯消去法**

為了求解 $Ax = b$ 形式的 $n$ 個方程式，消去法需要約 $\frac{1}{3}n^3$ 次乘法和減法。回代需要 $n^2$ 次運算。因此，總成本是 $\frac{1}{3}n^3 + n^2$。對於大的 $n$，$\frac{1}{3}n^3$ 項占主導地位。消去法需要約 $\frac{1}{3}n^3$ 次運算。

如果我們需要求解多個具有相同矩陣 A 的不同右側向量 b，則消去法仍然需要 $\frac{1}{3}n^3$ 次運算來分解 A。然後，對於每個 b，我們只需要 $n^2$ 次運算來進行回代。
