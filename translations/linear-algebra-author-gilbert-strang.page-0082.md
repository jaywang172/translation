**問題 25-31 使用列-行乘法或區塊乘法。**

**25** 將矩陣 A 乘以 I (3x3) 的行，其中 I 是單位矩陣。

**26** 將 AB 乘以行，使用列乘以行。

```
AB = [1 0] [3 0] = [1 3 0] + [2 4] [2 1] = [2 2]
```

**27** 證明兩個上三角矩陣的乘積總是上三角矩陣。

```
AB = [x x x] [x x x] = [0 0 0]
    [0 x x] [0 x x] = [0 0 0]
    [0 0 x] [0 0 x]
```

使用點積證明 (第 1 列乘以第 1 行) (第 2 列乘以第 2 行) (第 3 列乘以第 3 行)。哪些點積產生零？

使用有用矩陣 (列乘以行) 繪製 X 和 0 在 (第 2 列的第 2 行) (第 3 列的第 3 行) 的位置。

**28** 繪製切割線 (2x2) 和 (3x3) 以及 AB 的切割線，並顯示四種乘法規則的實際意義：

(1) 矩陣 A 的列乘以矩陣 B 的行。  列
(2) 矩陣 A 的行乘以矩陣 B 的列。  行
(3) 矩陣 A 的行乘以矩陣 B 的行。  內積 (AB 中的數字)
(4) 矩陣 A 的列乘以矩陣 B 的列。  外積 (矩陣加到 AB)

**29** 哪些矩陣 E<sub>21</sub> 和 E<sub>31</sub> 產生零在 (1, 2), (1, 3) 位置的 E<sub>21</sub>A 和 E<sub>31</sub>A？

```
A = [2 1 0]
    [-2 0 1]
    [8 5 3]
```

找到單一矩陣 E = E<sub>31</sub>E<sub>21</sub> 產生零在上述位置。將 E 乘以 A。

**30** 區塊乘法消去矩陣 E 的第 1 列。

```
EA = [1 0] [a b] = [a b]
    [-c/a 1] [c D] = [D - c/a]
```

在問題 29 中，什麼是 c 和 D，以及 D = -c/a？

**31** 以 i<sup>2</sup> = -1，將 (A + iB) 和 (x + iy) 乘以 (Ax + iAy) 。使用區塊分離實部和虛部，並從實部和虛部中分離出實部和虛部。

```
[A - B] [x y] = [Ax - By] (實部)
[C - D] [i i] = [Ay + Bx] (虛部)
```
