## 1.3. 矩陣 27

所有三個向量 **u**, **v**, **w** 的所有線性組合加總起來都等於零。正如我們之前所見，這表示 b₁ + b₂ + b₃ = 0 (這就是包含 **u** 和 **v** 的所有組合的平面方程式)。透過包含 **w**，我們得到了不再位於該平面的新向量。原始的 **w** = (0, 0, 1) 不在該平面上：0 + 0 + 1 ≠ 0。**u**, **v**, **w** 填滿了整個三維空間。我們已經知道，因為解 x = Sb 在方程式 (6) 中給出了產生向量 **b** 的正確線性組合。

這兩個矩陣 A 和 C，其第三欄分別為 **w** 和 **w***，讓我們得以提及線性代數的兩個關鍵詞：獨立性和依賴性。本課程的前半部分將深入探討這些概念，我很高興能用以下兩個範例來闡述它們：

*   **u**, **v**, **w** 彼此獨立。除了 0**u** + 0**v** + 0**w** = 0 之外，沒有其他線性組合能得到 **b** = 0。
*   **u**, **v**, **w*** 彼此依賴。某些線性組合 (特別是 c₁**u** + c₂**v** + c₃**w***) 得到 **b** = 0。

您可以想像在三維空間中，三個向量位於一個平面上或不在平面上。第 2 章討論了 n 個向量在 n 維空間中的情況。獨立性是關鍵。向量進入一個 n x b 矩陣的欄位。

獨立的欄位：Ax = 0 只有一個解。A 是一個可逆矩陣。
依賴的欄位：Ax = 0 有無限多個解。A 是一個奇異矩陣。

最終，我們將有 n 個向量在 m 維空間中。矩陣 A，其欄位數為 n，而列數為 m。理解 Ax = **b** 是第 3 章的問題。

### REVIEW OF THE KEY IDEAS

1.  矩陣乘以向量：Ax = 欄向量的線性組合。
2.  解方程式 Ax = **b** 是 x = A⁻¹**b**，當 A 是可逆矩陣時。
3.  差矩陣 C 被差矩陣 S 的反矩陣 S⁻¹ 反轉。
4.  循環矩陣 A 沒有反矩陣。其所有欄向量都在同一條線上。
    這些依賴的欄向量加總到零向量。Cx = 0 有無限多個解。

### WORKED EXAMPLES

1.3 A 改變了邊界條件 a₃₁ = 1 的 A (row 3, column 1) 的 1/3。

```
[ 1  0  0 ]   [ 1  0  0 ]
[ 0  2  0 ] → [ 0  2 -1 ]
[ 0  0  1 ]   [ 0  0  1 ]
```

改變此欄向量會如何影響解 x = A⁻¹**b**，其中 **b** = (1, 1, 1)？

解：A⁻¹**b** = **b** - **w**。由於 A⁻¹**b** = **b** - **w**，改變 a₃₁ 不會影響解 x。
