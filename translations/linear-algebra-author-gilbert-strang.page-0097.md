**45** 針對 2x2 區塊矩陣的消去法：當你將第一區塊列乘以 C⁻¹A⁻¹ 並從第二區塊列中減去時，“舒爾補集”S 會出現：

```
[ I   0 ] [ A  B ] = [ A  B ]
[-CA⁻¹ I ] [ C  D ] = [ 0  S ]
```

其中 A 和 D 是方陣，且 S = D - CA⁻¹B。

將右側乘以 A⁻¹B 以從第二區塊列中減去第一區塊列乘以 A⁻¹B 的區塊列。

```
[ A  B ] [ I  -A⁻¹B ] = ?
[ 0  S ] [ 0   I   ]
```

求 S，其中

```
[ A  B ] = [ 2  3  3 ]
[ C  I ] = [ 4  1  0 ]
         [ 4  0  1 ]
```

區塊樞軸是 A 和 S。如果它們可逆，則 so is [ A  B ; C  D ]。
