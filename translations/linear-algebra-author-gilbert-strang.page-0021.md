假設一個向量是另一個向量的倍數，即 **v** = c**w**。那麼角度為 0° 或 180°。 在這種情況下 cos θ = 1 且 |**v** ⋅ **w**| = ||**v**|| ||**w**||。 如果角度為 0°，如 **v** = 2**w**，則 ||**v** + **w**|| = ||**v**|| + ||**w**||。 三角形完全扁平。

**1.2 B** 找到一個在 **v** = (3, 4) 方向上的單位向量 **u**。 找到一個與 **u** 垂直的單位向量 **U**。 有多少個可能的 **U**？

**解題** 對於一個向量 **v**，除以其長度 ||**v**|| = 5。 為了得到一個垂直向量 **v**，我們選擇 (-4, 3)，因為點積 **v** ⋅ **V** = (3)(-4) + (4)(3) = 0。 對於一個向量 **V**，除以其長度 ||**V**||。

**u** = **v** / ||**v**|| = (3/5, 4/5)

**U** = **V** / ||**V**|| = (-4/5, 3/5)

**u** ⋅ **U** = 0

唯一的另一個垂直單位向量將是 -**U** = (4/5, -3/5)。

**1.2 C** 找到一個向量 **x** = (c, d) 使得點積 **x** ⋅ **r** = 1 且 **x** ⋅ **s** = 0，其中給定的向量 **r** = (2, -1) 且 **s** = (-1, 2)。

如何解決這個問題？ 參考範例 1.1 C，我們解出 c**v** + d**w** = **b** (1.0)?

**解題** 這些點積給出了關於 c 和 d 的線性方程式。 因此 **x** = (c, d)。

**x** ⋅ **r** = 1  2c - d = 1

**x** ⋅ **s** = 0  -c + 2d = 0  與範例 1.1 C 中的方程式相同。

第二個方程式給出了 **x** 垂直於 **s** 的資訊。 所以 **x** = (1, 2)。 根據幾何學，在垂直方向上，(2, 1)。 當你求解 **x** = ½(2, 1) 時，點積得到 **x** ⋅ **r** = (2, -1) 具有要求的性質。

評論：在方程式中將向量 **x** = (c, d) 表示為 **n** ⋅ **b** 的形式來解決問題。 在這種情況下，我們從 **r** 和 **s** 獲得 **n**。 為了得到 **x**，我們需要找到 **n** 的正交補集。 為了找到正交補集，我們需要找到 **n** 的零空間。 零空間是 **n** 的垂直空間。 在這種情況下，零空間是 **s** 的倍數。 因此，**x** = ½(2, 1) 是解。
