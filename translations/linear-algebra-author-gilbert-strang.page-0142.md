## 3.2. 矩陣 A 的零空間：解 Ax = 0

**回顧重點**

1.  零空間 N(A) 是 Rn 的一個子空間。它包含所有 Ax = 0 的解。
2.  消去法產生一個行階梯矩陣 U，然後再產生一個行簡化矩陣 R，其中包含樞軸列和自由列。
3.  U 或 R 的每個自由列都導向一個特解。自由變數等於 1，其他變數等於 0。回代法解 Ax = 0。
4.  Ax = 0 的完整解是特解的線性組合。
5.  如果 n > m，則 A 至少有一列沒有樞軸，從而產生一個特解。因此，存在非零向量 x 在這個矩陣 A 的零空間中。

**例題**

3.2 A 建立一個 3x4 矩陣 A，其 Ax = 0 的特解為 s1 和 s2：

s1 =  [-3, 1, 0, 0]ᵀ  和  s2 = [-2, 0, -6, 1]ᵀ

樞軸列為 1 和 3
自由變數為 x2 和 x4

您可以建立一個行簡化形式的矩陣 R。然後描述所有具有所需零空間 N(A) = 所有 s1 和 s2 的線性組合的矩陣 A。

**解答**

行簡化矩陣 R 有兩個樞軸，分別在第 1 列和第 3 列。因此，R 的第 2 列和第 4 列是樞軸列的線性組合：

R =  [1 3 0 2]
     [0 0 1 6]
     [0 0 0 0]

具有 Rs1 = 0 和 Rs2 = 0。

R 中的條目 3、2、6 和 -6 是特解 -3、-2、-6 的負數。我們可以對 R 進行任何基本行運算（交換行、將某行乘以一個非零常數、從一行減去另一行的倍數），而不會改變其零空間。矩陣 R 可以乘以（從左側）任何可逆矩陣，而不會改變其零空間。

每個 3x4 矩陣都有一個唯一的特解。這些矩陣都有兩個自由變數。
