# 結合子空間的基底

以下是一些關於基底的重要事實。我們將其保留至現在——直到我們準備好使用它們。經過一週後，你對於基底的定義應該有了更清晰的認識（線性獨立且能生成該空間的向量）。通常我們必須檢查這兩種性質。而當向量數量正確時，其中一種性質會推導出另一種性質：

$\mathbb{R}^n$ 中任何 $n$ 個獨立向量必定能生成 $\mathbb{R}^n$。因此它們是一個基底。
任何能生成 $\mathbb{R}^n$ 的 $n$ 個向量必定是獨立的。因此它們是一個基底。

從正確的向量數量開始，基底的一種性質可以產生另一種性質。這在任何向量空間中都成立，但我們最關心的是 $\mathbb{R}^n$。當這些向量成為一個 $n \times n$ 方陣 $A$ 的列（columns）時，同樣有以下兩個事實：

若 $A$ 的 $n$ 個列向量是獨立的，則它們能生成 $\mathbb{R}^n$。因此 $Ax = b$ 是可解的。
若 $n$ 個列向量能生成 $\mathbb{R}^n$，則它們是獨立的。因此 $Ax = b$ 只有唯一解。

唯一性隱含存在性，且存在性隱含唯一性。此時 $A$ 是可逆的。如果沒有自由變數，則解 $x$ 是唯一的。必須有 $n$ 個主元（pivots）。接著透過回代法可求解 $Ax = b$（解存在）。

從相反方向開始，假設對於每個 $b$，$Ax = b$ 都能求解（解的存在性）。那麼消去法不會產生零列（zero rows）。存在 $n$ 個主元且沒有自由變數。零空間（nullspace）僅包含 $x = 0$（解的唯一性）。

結合行空間（row space）與零空間的基底，我們共有 $r + (n - r) = n$ 個向量。這是正確的數量。這 $n$ 個向量是獨立的$^2$。因此它們能生成 $\mathbb{R}^n$。

每個 $x$ 都可以表示為行空間向量 $x_r$ 與零空間向量 $x_n$ 之和 $x_r + x_n$。
