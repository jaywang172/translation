## 3.5. 獨立性、基底與維度 177

**運算範例**

**3.5 A** 從向量 v₁ = (1, 2, 0) 和 v₂ = (2, 3, 0) 開始。 (a) 這些向量線性獨立嗎？ (b) 這些向量是否為任何空間的基底？ (c) 這些向量所張成的空間 V 是什麼？ (d) V 的維度是多少？ (e) 哪些矩陣 A 具有 V 作為其欄空間？ (f) 哪些矩陣具有 V 作為其零空間？ (g) 描述所有向量 v₃，使其與 v₁, v₂ 形成 R³ 的基底。

**解答**

(a) v₁ 和 v₂ 是線性獨立的——唯一能使 0 = θ₁v₁ + θ₂v₂ 的組合是 θ₁ = 0 和 θ₂ = 0。

(b) 是的，它們是基底，因為它們張成該空間。

(c) 該空間 V 包含所有向量 (x, y, 0)。它是 R³ 中的 xy 平面。

(d) V 的維度是 2，因為它包含兩個向量。

(e) V 是任何具有 3 個欄的矩陣 A 的欄空間，如果每個欄都是 v₁ 和 v₂ 的線性組合。特別是，A 可以只有欄 v₁ 和 v₂。

(f) V 是任何具有 3 個欄的矩陣 B 的零空間，如果每個欄都是 (0, 0, 1) 的倍數。特別是，B 可以是 [0 0 1]。那麼 Bv₁ = 0 且 Bv₂ = 0。

(g) 任何向量 v₃ = (a, b, c) 只要 c ≠ 0，就能完成 R³ 的基底。

**3.5 B** 從三個獨立向量 w₁, w₂, w₃ 開始。寫出產生 w₁, w₂, w₃ 的組合的矩陣形式，如 V = W M：

v₁ = w₁ + w₂
v₂ = w₁ + 2w₂ + w₃  which is  [v₁ v₂ v₃] = [w₁ w₂ w₃] [1 1 0; 1 2 1; 0 1 c]
v₃ = w₂ + cw₃

測試 V 中的矩陣 V 是否線性獨立。如果 c ≠ 1，則顯示 w₁, w₂, w₃ 是線性獨立的。如果 c = 1，則顯示 w₁, w₂, w₃ 是線性相依的。

**解答**

該測試的目的是為了任何線性獨立的其欄的矩陣。零空間必須只包含零向量。
零空間 V 的欄必須包含零向量。因此，如果 c = 1，則 v₃ 是 v₁ 和 v₂ 的線性組合。
在這種情況下，如果 v₃ = v₁ + v₂，則 v₃ = (w₁ + w₂) + (w₁ + 2w₂ + w₃) = 2w₁ + 3w₂ + w₃，這與 v₃ = w₂ + w₃ 不符。
因此，如果 c ≠ 1，則 w₁, w₂, w₃ 是線性獨立的。如果 c = 1，則 w₁, w₂, w₃ 是線性相依的，因為 v₃ = w₂ + w₃ = (w₁ + w₂) + (w₁ + 2w₂ + w₃) = 2w₁ + 3w₂ + w₃，這表示 v₃ 可以用 v₁ 和 v₂ 表示。
