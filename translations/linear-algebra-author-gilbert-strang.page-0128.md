## 3.1. 向量空間

125

柱向量空間是所有這些向量的線性組合所填滿的平面在 R³ 中。我們繪製出特定 b（一個柱向量的組合）。此 b = Ax 軸在平面上。此平面沒有零厚度，因此在 R³ 中大多數右側在兩個方程式中沒有解。

當然 (0, 0) 在柱向量空間中。此平面始終通過原點。存在一個解 Ax = 0。此解始終可用，即 x = 0。

為了重複，可達到的右側 b 正確地是柱向量空間中的向量。一種可能性是第一列的右側，即取 x₁ = 1 且 x₂ = 0。另一種組合是第二列，即取 x₁ = 0 且 x₂ = 1。新的理解層次是理解所有線性組合——整個子空間由這些向量生成。

符號 柱向量空間可以記為 C(A)。從柱和取它們的線性組合。我們可能得到整個 Rⁿ 或一個子空間。

重要 重要的是，我們可以從向量集合 S 開始，在向量空間 V 中，我們可以從該集合中的向量開始。

S = S 的向量集合（可能不是子空間）
SS = S 的所有線性組合
SS = all c₁v₁ + … + cₙvₙ  — 由 S “張成” 的 V 的子空間

當 S 是向量集合時，SS 是柱向量空間。當 S 中的只有一個非零向量 v 時，SS 是通過 v 的線。總是包含 S 的最小子空間。這是創建子空間的基本方法，我們稍後會回到它。

子空間 SS 是 S “張成” 的，包含 S 的所有線性組合。

範例 5 描述柱向量空間（它們是 R² 的子空間）

```
I = [ 1 0 ]   and   A = [ 1 2 ]
    [ 0 1 ]           [ 0 4 ]
```

```
B = [ 1 2 ]
    [ 0 4 ]
```

解 I 的柱向量空間是整個 R²。每個向量都是 I 的柱的組合。

A 的柱向量空間是線。C(A) 是 R²。每個向量都是 A 的第一列 (1, 0) 的倍數。或者，它是 (1, 2) 和 (0, 4) 的線性組合。注意 (1, 2) 和 (0, 4) 都在同一條直線上，因此它們的線性組合也都在同一條直線上。

B 的柱向量空間是線。C(B) 是 R²。每個向量都是 B 的第一列 (1, 0) 的倍數。或者，它是 (1, 2) 和 (0, 4) 的線性組合。注意 (1, 2) 和 (0, 4) 都在同一條直線上，因此它們的線性組合也都在同一條直線上。

總之，如果矩陣的列位於同一條直線上，則 C(A) 是一條線。如果矩陣的列位於整個平面上，則 C(A) 是整個 R²。
