## 第 3 章
### 向量空間與子空間

#### 3.1 向量的空間

對於一個新手來說，矩陣計算涉及大量的數字。對你而言，它們涉及向量。
*Ax* 和 *AB* 的列是 *n* 個向量——*A* 的列——的線性組合。本章從數字和向量到第三層理解的提升（最高層次）。
與觀察個別向量不同，我們關注“向量的空間”，特別是它們的子空間。如果你沒有理解 *Ax = b*，那也沒關係。
既然本章會深入一點，可能看起來有點困難。這很自然。我們正在尋找內在的數學。作者的工作是使其清晰。本章的結論是“線性代數的基本定理”。

我們從最重要的向量空間開始。它們被表示為 **R**<sup>1</sup>, **R**<sup>2</sup>, **R**<sup>3</sup>, **R**<sup>4</sup>, ...。每個空間 **R**<sup>n</sup> 由 *n* 個分量的向量組成。**R**<sup>5</sup> 包含具有五個分量的所有向量。這被稱為“五維空間”。

**定義** 空間 **R**<sup>n</sup> 由具有 *n* 個分量的所有列向量 **v** 組成。

**v** 的分量是實數，這就是字母 **R** 的原因。具有 *n* 個複數分量的向量位於空間 **C**<sup>n</sup> 中。

向量空間 **R**<sup>2</sup> 由通常的 *xy* 平面表示。**R**<sup>2</sup> 中的每個向量 **v** 都有兩個分量。單詞“空間”要求我們思考所有這些向量——整個平面。每個向量給出平面上的點的座標：**v** = ( *x*, *y* )。

同樣地，**R**<sup>3</sup> 中的向量對應於三維空間中的點 ( *x*, *y*, *z* )。
一維空間 **R**<sup>1</sup> 是一條線（如 *x* 軸）。正如之前所見，我們將向量寫成列向量，用方括號括起來：

[4] 在 **R**<sup>2</sup> 中，(1, 0, 1, 1) 在 **R**<sup>5</sup> 中，[1+i] 在 **C**<sup>2</sup> 中。
[π]                                  [1-i]

線性代數的偉大之處在於它能輕鬆處理五維空間。我們不需要繪製向量，我們只需要五個數字（或 *n* 個數字）。
