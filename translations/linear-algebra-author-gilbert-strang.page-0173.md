**第 3 章：向量空間與子空間**

在 R² 中，三個向量無法獨立。一種檢視方式是：矩陣 A 與這些向量作為列向量時，會有一個自由變數，且方程式 Ax = 0 有非零解。如果前兩個向量獨立，那麼剩餘的向量必須是它們的線性組合。

現在讓我們考慮在 R³ 中的三個向量。如果其中一個是另一個的倍數，那麼這些向量是相依的。但完整的測試涉及所有三個向量，我們將它們放入矩陣中並解 Ax = 0。

**範例 1**

矩陣的列向量 Ax = 0 有一個非零解。

```
[ 1  0  3 ]   [ -3 ]
[ 2  1  5 ] * [  1 ] = [ 0 ]
[ 1  0  3 ]   [  1 ]   [ 0 ]
```

此矩陣的秩為 r = 2。獨立的列產生完整的列秩 r = n = 3。

在這個矩陣中，列向量也是相依的。Row 1 minus Row 3 is the zero row。對於一個方陣，我們將展示相依的列向量意味著相依的列向量。

**問題** 如何找到使得 Ax = 0 的解？系統性的方法是消去法。

```
A = [ 1  0  3 ]   
    [ 2  1  5 ]   
    [ 1  0  3 ]   
```

化簡為

```
R = [ 1  0  3 ]   
    [ 0  1 -1 ]   
    [ 0  0  0 ]   
```

解 x = (-3, 1, 1) 恰好是特殊解。它顯示了自由列（Column 3）是樞軸列的組合。這殺死了獨立性！

**完整的列秩** 矩陣 A 的列向量獨立，當且僅當秩 r = n。

這表示 n 個樞軸且沒有自由變數。只有 x = 0 在零空間中。

這點至關重要，因為一開始就已經很清楚了。假設七個向量有五個分量（m = 5 小於 n = 7）。那麼這些列向量必須相依。任何七個向量來自 R⁵ 是相依的。秩 r 至少小於 5。不能有超過五個樞軸。Ax = 0 有至少 7 – 5 = 2 個自由變數，因此它有無限多個解——這意味著列向量是相依的。

**任何 n 個向量在 Rᵐ 空間中，如果 m < n，則線性相依。**

一個方陣有完整的列秩，如果它的列向量獨立。列向量是線性相依的，如果存在一個非零解 x，使得 Ax = 0。

在矩陣 A 中，如果 r < n，則存在自由變數。這意味著 Ax = 0 有無限多個解。這些解形成零空間。零空間的維度是 n – r。零空間包含所有與 A 相容的向量。換句話說，零空間包含所有可以被消去以產生行梯形形式的向量（也就是說，向量 x 使得 Ax = 0）。零空間是矩陣 A 的核（kernel）。
