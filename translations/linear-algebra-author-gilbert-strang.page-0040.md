## 2.1. 向 量 與 線性 方程

**範例 1** 這裡有 3x3 的矩陣 *A* 和 *I*，*I* 是單位矩陣，包含三個 1 和六個 0：

```
A = [ 1 0 0 ]   [ 4 ]     I = [ 1 0 0 ]
    [ 0 1 0 ] * [ 4 ] =   [ 0 1 0 ]
    [ 0 0 1 ]   [ 6 ]     [ 0 0 1 ]
```

如果 *x* 是一個向量，那麼線性組合 *Ax* 等於 (1, 0, 0) 與 (4, 5, 6) 的線性組合。如果 *x* 是一個列向量，那麼 *Ax* 是 *A* 的第一列乘以 4。在矩陣 *A* 中，第二和第三列是零向量。

另一個矩陣 *I* 是特殊的。它在「主對角線」上有 1，其餘位置為 0。任何向量乘以這個矩陣，該向量不會改變。這就像乘以 1，但對於矩陣和向量而言。這個特殊的矩陣在這個例子中是 3x3 的單位矩陣：

```
Ix = x
```

總是產生這個乘法結果。

### 矩陣 標記法

第一個 2x2 的矩陣包含 *a₁₁* 和 *a₁₂*。第二行包含 *a₂₁* 和 *a₂₂*。第一個索引給定行數，因此 *aᵢⱼ* 是第 *i* 行第 *j* 列的元素。第二個索引給定列數。但使用下標表示法不一定很方便！相反，我們寫 *A(i, j)*。條目 *a₅₇* 等於 *A(5, 7)*，表示第 5 行第 7 列。

對於一個 *m* x *n* 的矩陣，行索引從 1 到 *m*。列索引從 1 到 *n*。總共有 *mn* 個條目 *aᵢⱼ* = *A(i, j)*。一個正方形矩陣有 *n²* 個條目。

### MATLAB 中的乘法

我想表達 *x* 和 *Ax* 的結果，並使用 MATLAB 命令來學習。這是一個學習 *A* 和 *x* 的第一步，首先定義矩陣 *A* 和向量 *x*。這個向量是一個 3x1 的矩陣，其中包含一行和三列。輸入矩陣和向量時，使用分號來表示新的一行：

```
A = [ 1 2 3; 2 5 2; 6 -3 1 ]
x = [ 0; 0; 2 ]
```

現在讓我們在 MATLAB 中執行 *Ax*。*A* 乘以 *x* 的結果是：

```
MATLAB 是一個強大的工具，可以執行矩陣運算。
```

```
Ax = A * x
```
