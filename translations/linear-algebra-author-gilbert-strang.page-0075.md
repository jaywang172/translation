第 2 章 求解線性方程組

這是常規消去法，一列一次地以區塊形式書寫。最終的區塊 *S* 是 *D* − *CA*⁻¹*B*，恰好是舒爾補集。

**■ 關鍵概念回顧 ■**

1.  矩陣 *AB* 的 ( *i*, *j* ) 入口是 ( *A* 的第 *i* 列 ) ⋅ ( *B* 的第 *j* 行 )。
2.  一個 *m* × *n* 矩陣乘以一個 *n* × *p* 矩陣需要 *mnp* 次獨立的乘法。
3.  *A* 乘以 *BC* 等於 *AB* 乘以 *C* (令人驚訝地重要)。
4.  *AB* 也可以表示為以下形式：( *A* 的各列 ) 乘以 ( *B* 的各行 )。
5.  區塊乘法成立的前提是區塊匹配正確。
6.  消去法產生舒爾補集 *D* − *CA*⁻¹*B*。

**■ 練習範例 ■**

2.4 A 讓我們把自己置於作者的位置！我想向你展示一些特殊的矩陣乘法，但大多數時候我們都會遇到小矩陣。它們具有令人驚嘆的特性，並且以各種大小和形式出現，而且都具有相同的含義。我想用 4 × 4 的下三角矩陣 *L* 來展示這些模式。它們驚人的模式來自“帕斯卡三角形”。我將 *L* 乘以一個向量和冪向量：

```
Pascal
matrix
[ 1 ]
[ 1  1 ]
[ 1  2  1 ]
[ 1  3  3  1 ]
=
[ 1 ]
[ 2 ]
[ 4 ]
[ 8 ]
```

```
[ 1 ]
[ 1  1 ]
[ 1  2  1 ]
[ 1  3  3  1 ]
*
[ 1 ]
[ x ]
[ x² ]
[ x³ ]
=
[ 1 ]
[ 1 + x ]
[ 1 + 2x + x² ]
[ 1 + 3x + 3x² + x³ ]
```

*L* 的每一行都通過在它的左邊添加 1 來獲得下一行。在符號 *lᵢ,ⱼ* = *lᵢ₋₁,ⱼ* + *lᵢ,ⱼ₋₁* 中，數字在 1, 3, 3, 1 之後，將會是 1, 4, 6, 4, 1。帕斯卡在 1600 年代，在研究矩陣之前，就已經研究了帕斯卡三角形，但它的三角形模式與 *L* 完全匹配。

將 *L* 乘以 1 的向量，相當於沿著每一行進行冪展開。通過將 *L* 乘以 *x* 的向量，我們可以得到“二項式係數”，它們對於冪展開至關重要：

1 + 2*x* + *x*² = (1 + *x*)²   1 + 3*x* + 3*x*² + *x*³ = (1 + *x*)³

這個數字“3”計算了拋擲硬幣兩次時正面和反面的組合數量：HHT 和 THT。另一個“3”計算了拋擲硬幣三次時正面和反面的組合數量：HHT 和 THH。
