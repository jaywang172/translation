**FORTRAN 程式範例**

以下程式範例示範如何計算兩個向量的內積，以及如何計算向量的長度。請注意，FORTRAN 使用從 1 開始的陣列索引。

**範例 7** 假設 `V(1)` 到 `V(N)` 包含向量 **v** 的分量，而 `W(1)` 到 `W(N)` 包含向量 **w** 的分量。以下程式碼計算內積 `V · W` 並將結果儲存在 `DOT` 中，並計算長度 `||v||` 並將結果儲存在 `VLEN` 中。

```fortran
      DIMENSION V(100), W(100)
      REAL DOT, VLEN
      INTEGER I, N

      N = 10  ! 向量的維度

      ! 計算內積
      DOT = 0.0
      DO 10 I = 1, N
         DOT = DOT + V(I) * W(I)
 10    CONTINUE

      ! 計算 ||v||
      VLEN = 0.0
      DO 20 I = 1, N
         VLEN = VLEN + V(I) * V(I)
 20    CONTINUE
      VLEN = SQRT(VLEN)

      PRINT *, '內積 V · W =', DOT
      PRINT *, '||v|| =', VLEN
```
