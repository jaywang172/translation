## 2.3. 使用矩陣進行消去法 57

一點需要重複說明的是矩陣標記法：第一列第一行（左上角）的元素是 a₁₁. 第一列第三行（右下角）的元素是 a₃₁. 矩陣中的數字（行號和列號）被稱為“分量”或向量。一般規則是：aᵢⱼ = A(i, j)，i 是第 i 行，j 是第 j 列。

**例題 1** 這個矩陣有 a₁₂ = 2i + j。那麼 a₁₃ = 3。另外 a₁₂ = 4 且 a₂₁ = 5。
這裡 Ax 有數字和字母：

[ 3  4  2 ] [ 1 ] = 3・2 + 4・1
[ 5  6  1 ] [ x₂ ] = 5・2 + 6・1

[ a₁₁ a₁₂ ] [ x₁ ] = [ a₁₁x₁ + a₁₂x₂ ]
[ a₂₁ a₂₂ ] [ x₂ ] = [ a₂₁x₁ + a₂₂x₂ ]

這個矩陣的第一個分量是 6 + 4 = 10。一個行乘以一個向量給出一個點積。

**矩陣形式的單步消去法**

Ax = b 是一個方便的原始方程式形式。消去法的第一步在這個例子中從第二個方程式中減去第一個方程式的兩倍。在右側，第二個分量的兩倍被從第一個分量中減去。

**第一步**

b = [ 2 ] 變成 bnew = [ 2 ]
      [ 8 ]                [ 4 ]
      [ 10]                [ 10]

我們想要通過一個矩陣！這個相同的結果 bnew = Eb 被實現，當我們將一個“消去矩陣”E 乘以 b₂ 時，它從 b₂ 中減去 2b₁。

消去矩陣是 E = [ 1  0  0 ]
                   [ -2 1  0 ]
                   [ 0  0  1 ]

將 E 乘以 b 會從第一行減去 1 倍，第二行減去 2 倍。行 1 和行 3 保持不變：

[ 1  0  0 ] [ 2 ] = [ 2 ]
[ -2 1  0 ] [ 8 ] = [ -4 + 8 ] = [ 4 ]
[ 0  0  1 ] [ 10] = [ 10]

這個矩陣 E 的第一行和第三行是單位矩陣。新的第二個分量是 4，它出現在這裡，因為它來自 2b₁。
很容易描述這些“基本矩陣”或“消去矩陣”的特性。從單位矩陣 1 開始。改變任何它的元素為零。這個規則是 -1。

**消去矩陣的特性**

A 的消去矩陣 E 具有以下特性：
1.  E 乘以 A 的每一行給出 A 的新行。
2.  E 乘以 b 產生 b 的新向量。
3.  消去矩陣 E 是可逆的。
