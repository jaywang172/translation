## 2.2. 消去法的概念 51

**2.2 C** 使用消去法將矩陣化為上三角矩陣 U。使用回代法求解，或解釋為何這是不可能的。當樞元為零時，交換列。交換的唯一差異在於最後一列的符號。

**成功**  x + y + z = 7
**則**   x + y - z = 5
**失敗** x + y + z = 3

**解** 對於第一個系統，我們從方程式 2 和 3 中減去方程式 1（乘數為 a<sub>21</sub> = 1 和 a<sub>31</sub> = 1）。這樣做會使方程式 2 變成零，因此我們交換方程式：

**成功**  x + y + z = 7
          0y - 2z = -4  交換成  -2y + 0z = -4
          0y - 2z = -4          -2z = -4

然後回代法給出 z = 1 和 y = 2，而 x = 4。樞元為 1、-2、-2。

對於第二個系統，我們從方程式 2 中減去方程式 1，如前所述。將方程式 1 加到方程式 3。這會留下以下方程式：

**失敗**  x + y + z = 7
          0y - 2z = -4
          0y + 2z = 10

存在一個樞元在第 2 列（等於第 1 列）。進一步的消去步驟會導致 0 = 8。
這三條平面不相交。

如果我們將“3”更改為原始第三方程式中的“-5”，則消去法會導致 0 = 0。存在無限多個解。這三條平面現在相交於一條直線。
將 3 更改為 -5 將第三條平面移動到與其他兩個平面相交。第二個方程式給出 z = 1。然後第一個方程式變成 x + y = 6。沒有樞元，因此 y 是自由的（它可以有任何值）。那麼 x = 6 - y。
