## 3.1. 向量空間

126

最重要且最常見的子空間是來自線性方程式的解集合。例如，考慮方程式 $ax + by = 0$。在平面上，這代表一條通過原點的直線。

**範例 5：** 考慮方程式 $x + y = 0$。此方程式的解集合是直線 $y = -x$。
*   (i) 零向量 $(0,0)$ 顯然是解。
*   (ii) 如果 $(x_1, y_1)$ 和 $(x_2, y_2)$ 是解，則 $x_1 + y_1 = 0$ 和 $x_2 + y_2 = 0$。因此，對於任何常數 $c$，
    $c(x_1, y_1) = (cx_1, cy_1)$ 滿足 $cx_1 + cy_1 = c(x_1 + y_1) = c(0) = 0$。
    此外，$(x_1, y_1) + (x_2, y_2) = (x_1 + x_2, y_1 + y_2)$ 滿足 $(x_1 + x_2) + (y_1 + y_2) = (x_1 + y_1) + (x_2 + y_2) = 0 + 0 = 0$。
    因此，此直線是一個子空間。

**範例 6：** 考慮方程式 $x - y = 0$。此方程式的解集合是直線 $y = x$。這也是一個子空間。

**範例 7：** 考慮方程式 $x + y = 1$。此方程式的解集合是一條直線，但不通過原點。因此，它不是一個子空間（因為它不包含零向量）。

**範例 8：** 在 $\mathbb{R}^3$ 中，考慮方程式 $x + y + z = 0$。此方程式的解集合是一個平面通過原點。這是一個子空間。

**範例 9：** 在 $\mathbb{R}^3$ 中，考慮方程式 $x + y + z = 1$。此方程式的解集合是一個平面，但不通過原點。因此，它不是一個子空間。

**定理：** 任何通過原點的平面或直線都是一個子空間。

**證明：** 讓我們考慮通過原點的平面 $ax + by + cz = 0$。
*   (i) 零向量 $(0,0,0)$ 滿足方程式。
*   (ii) 如果 $(x_1, y_1, z_1)$ 和 $(x_2, y_2, z_2)$ 是解，則 $ax_1 + by_1 + cz_1 = 0$ 和 $ax_2 + by_2 + cz_2 = 0$。
    對於任何常數 $c$，
    $c(x_1, y_1, z_1) = (cx_1, cy_1, cz_1)$ 滿足 $a(cx_1) + b(cy_1) + c(cz_1) = c(ax_1 + by_1 + cz_1) = c(0) = 0$。
    此外，$(x_1, y_1, z_1) + (x_2, y_2, z_2) = (x_1 + x_2, y_1 + y_2, z_1 + z_2)$ 滿足 $a(x_1 + x_2) + b(y_1 + y_2) + c(z_1 + z_2) = (ax_1 + by_1 + cz_1) + (ax_2 + by_2 + cz_2) = 0 + 0 = 0$。
    因此，此平面是一個子空間。

**結論：** 任何線性方程式的解集合（通過原點）都是一個子空間。

**重要提示：** 為了驗證一個集合是否為子空間，您必須驗證 (i) 和 (ii)。
