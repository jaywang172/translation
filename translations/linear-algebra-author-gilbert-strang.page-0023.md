## 20 章：向量的介紹

**問題 18-31 涉及三角形中的長度與角度。**

**18** 具有邊向量 **v** = (4, 2) 和 **w** = (-1, 2) 的平行四邊形是一個矩形。檢查畢氏定理 a² + b² = c² 是否僅適用於直角三角形。

(長度 **v**)² + (長度 **w**)² = (長度 **v** + **w**)²

**19** (向量點積的性質) 這些等式很有用：

(1) **v** · **w** = **w** · **v**  (2) **u** · (**v** + **w**) = **u** · **v** + **u** · **w**  (3) (**cv**) · **w** = c(**v** · **w**)

使用 (2) 證明 **u** · (**v** + **w**) = ||**u**|| ||**v**|| cos θ

**20** “餘弦定律”來自：

Cosine Law  ||**v** - **w**||² = ||**v**||² + ||**w**||² - 2 ||**v**|| ||**w**|| cos θ

如果 θ > 90° 證明 ||**v**||² + ||**w**||² < ||**v** - **w**||² (第三邊的平方)。

**21** 三角形不等式來自：||**v** + **w**|| ≤ ||**v**|| + ||**w**|| (向量的長度之和)。

問題 19 證明 ||**v**|| - ||**w**|| ≤ ||**v** - **w**|| (向量的長度之差)。

三角形不等式也可以被表述為：||**v**|| - ||**w**|| ≤ ||**v** - **w**|| ≤ ||**v**|| + ||**w**||

**22** 向量 **v** 與向量 **w** 的夾角 θ 滿足 cos θ = (**v** · **w**) / (||**v**|| ||**w**||)。

如果 **v** · **w** = 0，則 **v** 與 **w** 正交。如果 **v** · **w** > 0，則 θ 是銳角。如果 **v** · **w** < 0，則 θ 是鈍角。
