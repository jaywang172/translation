## 172

**第 3 章：向量空間與子空間**

**例題 5** 描述矩陣 *A* 的列空間與列空間。

```
A =  [ 1 4 ]
     [ 2 7 ]
     [ 3 5 ]
```

且 *A<sup>T</sup>* = 

```
[ 1 2 3 ]
[ 4 7 5 ]
```

這裡 *m* = 3 且 *n* = 2。

*A* 的列空間是 *R<sup>3</sup>* 中由 *A* 的兩列所張成的平面。*A* 的列空間是 *R<sup>2</sup>* 中由 *A<sup>T</sup>* 的列所張成的空間。請記住：行在 *R<sup>n</sup>* 中張成列空間。列在 *R<sup>m</sup>* 中張成列空間。相同的數字，不同的向量，不同的空間。

**向量空間的基底**

兩個向量無法張成整個 *R<sup>3</sup>*，即使它們是獨立的。四個向量無法張成 *R<sup>3</sup>*，即使它們張成 *R<sup>3</sup>*。我們想要足夠的獨立向量來張成空間（不多也不少）。“基底”就是正確的。

**定義 基底**

向量空間的基底是一個具有以下兩個性質的向量序列：

*   基底向量是線性獨立的。
*   基底向量張成該空間。

這個性質的組合在線性代數中是根本性的。向量空間中的每個向量都可以表示為基底向量的線性組合。與產生 *v* 的線性組合不同，因為基底向量 *v<sub>1</sub>*, ..., *v<sub>n</sub>* 是獨立的。

只有一種方法可以將 *v* 表示為基底向量的線性組合。

**理由：** 假設 *v* = *a<sub>1</sub>v<sub>1</sub>* + ... + *a<sub>n</sub>v<sub>n</sub>* 且 *v* = *b<sub>1</sub>v<sub>1</sub>* + ... + *b<sub>n</sub>v<sub>n</sub>*。通過減法，(*a<sub>1</sub>* - *b<sub>1</sub>*)*v<sub>1</sub>* + ... + (*a<sub>n</sub>* - *b<sub>n</sub>*)*v<sub>n</sub>* = 0。由於 *v<sub>1</sub>*, ..., *v<sub>n</sub>* 的獨立性，對於所有 *i*，*a<sub>i</sub>* - *b<sub>i</sub>* = 0。因此 *a<sub>i</sub>* = *b<sub>i</sub>*，且沒有兩種方法可以產生 *v*。

**例題 6** 

```
[ 1 0 ]
[ 0 1 ]
```

的列產生 *R<sup>2</sup>* 的“標準基底”。

基底向量 **i** = 

```
[ 1 ]
[ 0 ]
```

和 **j** = 

```
[ 0 ]
[ 1 ]
```

是獨立的。它們張成 *R<sup>2</sup>*。

每個人都先想到單位矩陣。單位矩陣的 3 x 3 單位矩陣產生標準基底 *i*, *j*, *k*。單位矩陣的 *n* x *n* 單位矩陣產生“標準基底”為 *R<sup>n</sup>*。

注意，給定的基底不一定是唯一的！
