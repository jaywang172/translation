**第二章：解線性方程式**

你會看到乘數規則，如果我將第一個方程式變更為 4x - 8y = 4。（同樣地，直覺上你會看到第一個主項變成 4。）正確的乘數是 now l = 4。為了找到乘數，將係數 “3” 除以要被消除的主項 “4”：

```
4x - 8y = 4       乘以方程式 1 的 ⅓   4x - 8y = 4
3x + 2y = 11      從方程式 2 減去       8y = 8
```

最終的方程式是三角形式，最後一個方程式給了 y = 1。回代法產生 4x - 8 = 4 且 4x = 12 且 x = 3。我們已經得到了數字，但回代法並不是解的唯一方法。除以主項以找到乘數 l = ⅓。

```
主項 = 執行消除的列中的第一個非零項
乘數 = (要消除的項) / (主項) = ⅓
```

第二個新的方程式以主項 8 開始。我們需要從第三個方程式中消除主項，以解決 n 個方程式，我們需要 n 個主項。主項位於三角形的對角線上。

你可能已經解決了這些方程式，而不需要閱讀這本書。這是一個內在的笨拙問題，但我們將以小而緊湊的例子來解決。對於一個 2x2 系統，消除可能會崩潰。透過了解可能的崩潰（當我們無法找到主項時），你將理解整個消除過程。

## 崩潰的消去法

通常，消去法會產生主項，以便解決方程式。但失敗是可能的。有時，該方法會要求我們除以零。我們無法做到。該過程必須停止。可能存在調整和繼續——或失敗的不可避免的。

範例 1 失敗，因為沒有解，滿足 0y = 8。範例 2 失敗，因為有太多解，滿足 0y = 0。範例 3 成功地交換了方程式。

**範例 1**  永久性失敗，無解。消去法如下：

```
x - 2y = 1       從方程式 3 減去   x - 2y = 1
3x - 6y = 11      方程式 2          0y = 8
```

沒有解，滿足 0y = 8。通常，我們說沒有解。解決方程式的重點不在於找到答案，而是在於了解為什麼沒有答案。（沒有人要求你解決方程式！）。如果我們以不同的方式來處理，我們可能會得到一個不同的答案。

這個範例顯示了我們在嘗試解決線性方程式時可能遇到的問題。如果我們在消除過程中遇到 0y = b（其中 b ≠ 0），則表示該系統無解。

**範例 2**  永久性失敗，無數解。消去法如下：

```
x - 2y = 1       從方程式 2 減去   x - 2y = 1
3x - 6y = 3       方程式 2          0y = 0
```

現在我們有無限多個解。x - 2y = 1 描述了一條線。3x - 6y = 3 描述了同一條線。它們是相同的方程式。

**範例 3**  成功地交換方程式。

```
x - 2y = 1
3x - 6y = 11
```

交換方程式 2 和方程式 1：

```
3x - 6y = 11
x - 2y = 1
```

現在，從方程式 1 減去 3 倍的方程式 2：

```
3x - 6y = 11
-3x + 6y = -3
```

結果是 0 = 8，這表示該系統無解。
