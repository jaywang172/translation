## 126 章 3. 向量空間與子空間

**■ 關鍵概念回顧 ■**

1.  Rⁿ 包含 n 個實數向量。
2.  M (2 by 2 矩陣) 和 F (函數) (零向量單獨) 是向量空間。
3.  包含 v 和 w 的子空間及其所有線性組合 cv + dw。
4.  A 的欄向量的線性組合形成欄空間 C(A)。欄空間是“張成”的由欄向量。
5.  Ax = b 有解，當且僅當 b 在 A 的欄空間中。

**■ 練習範例 ■**

3.1 A 我們給定三個不同的向量 **b₁**, **b₂**, **b₃**。構造一個矩陣 A，使得方程式 Ax = **b₁** 和 Ax = **b₂** 可解，但 Ax = **b₃** 無解。如何決定這是否可行？如何構建 A？

解：我們希望 **b₁** 和 **b₂** 在 A 的欄空間中，而 **b₃** 不在。最快速的方法是將 **b₁** 和 **b₂** 作為 A 的兩欄。那麼解就是 x = (1, 0) 和 x = (0, 1)。

此外，我們不希望 Ax = **b₃** 可解。因此不要使欄空間更大！僅保留 **b₁** 和 **b₂** 的欄，問題是：

Is Ax = [ **b₁** **b₂** ] [ x₁ ] = **b₃** 可解？  Is **b₃** 是 **b₁** 和 **b₂** 的線性組合？
                                  [ x₂ ]

如果答案是否定的，我們就有所需的矩陣 A。如果答案是肯定的，那麼不可能構建 A。當欄空間包含 **b₁** 和 **b₂** 時，它將不得不包含它們的所有線性組合。所以 **b₃** 必然在那個欄空間中，並且 Ax = **b₃** 必然可解。

3.1 B 描述向量空間 V 的一個子空間 S，然後描述 S 的一個子空間 SS。

V₁ = 所有 (1, 1, 0) 和 (1, 1, 1) 和 (1, 1, 1) 的線性組合。
V₂ = 所有垂直於 **u** = (1, 2, 1) 的向量，因此 **u** · **v** = 0。
V₃ = 所有對稱 2 by 2 矩陣 (M 的一個子空間)。
V₄ = 所有微分方程式 d²y/dx² = 0 的解 (F 的一個子空間)。

描述每個 V 兩次方式：所有線性組合，所有方程的解。
