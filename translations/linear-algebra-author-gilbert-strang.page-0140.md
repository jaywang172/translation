### 行階梯矩陣

前向消去法從 A 轉換到 U。它通過行運算來完成，包括行交換。它會繼續到下一個欄位中沒有主元可用的當前欄位。m × n “階梯” U 是一個階梯矩陣。

這裡有一個 4 × 7 階梯矩陣，其中三個主元 p 以粗體顯示：

U = [ p × × × × × × ]
    [ 0 p × × × × × ]
    [ 0 0 0 0 p × × ]
    [ 0 0 0 0 0 0 0 ]

三個主變數 x₁, x₂, x₆
四個自由變數 x₃, x₄, x₅, x₇
四個特殊解在 N(U) 中

**問題** 對於這個矩陣，欄空間是什麼，零空間是什麼？

**答案** 欄向量具有四個分量，因此它們位於 R⁴ (在 R¹¹) 中。第四個分量對於每個欄向量來說都是零。任何欄向量的組合——每個向量在欄空間中——都會產生一個分量為零的向量。因此，我們可以通過替換來求解 Ux = b。這些向量 b 是七個欄向量的所有可能組合。

零空間 N(U) 是一個 R⁷ 的子空間。Ux = 0 的解是所有特殊解的組合——對於每個自由變數：

1. 欄 3, 4, 5, 7 沒有主元。因此，自由變數為 x₃, x₄, x₅, x₇。

2. 將三個自由變數設為 0，並將其他自由變數設為零。

3. 求解 Ux = 0 對於主變數 x₁, x₂, x₆。

4. 這給出了零空間中的四個特殊解。

非零行在階梯矩陣中向下排列。主元是這些行中第一個非零條目。零的行數等於主元的數量。假設 A 有 m 個主元欄，則 R 的行數為 m，因此 A 至少有一個自由變數。系統 Ax = 0 總是具有至少一個解，因為 R 的最後一行是零。

零空間 N(U) 由四個特殊解張成（在向量空間中）。零空間中所有可能的向量都是這些特殊解的線性組合。
