**12** 令 *P* 為在 R³ 中的平面，其方程式為 *x + y - z = 4*。原點 (0, 0, 0) 不在 *P* 中！找出兩個在 *P* 中的向量，並檢查它們的和是否在 *P* 中。

**13** 令 *P₀* 為與前一個平面 *P* 平行，且通過 (0, 0, 0) 的平面。*P₀* 的方程式是什麼？找出兩個在 *P₀* 中的向量，並檢查它們的和是否在 *P₀* 中。

**14** R³ 的子空間是平面、直線、R³ 本身，或包含 (0, 0, 0) 的 Z。

   **(a)** 描述 R² 的所有子空間。
   **(b)** 描述 D 的所有子空間，D 是 2x2 對角矩陣的空間。

**15** **(a)** 穿過 (0, 0, 0) 的兩個平面的交集很可能是一個 ____，但它也可能是 ____。
   **(b)** 穿過 (0, 0, 0) 的一個平面與一條通過 (0, 0, 0) 的直線的交集很可能是一個 ____，但它也可能是 ____。
   **(c)** 如果 *S* 和 *T* 是 R⁵ 的子空間，那麼它們的交集 *S ∩ T* 是一個子空間嗎？這裡 *S ∩ T* 由同時在兩個子空間中的向量組成。檢查 *x + y* 和 *c.x* 的要求。

**16** 假設 *P* 是穿過 (0, 0, 0) 的一個平面，而 *L* 是一條穿過 (0, 0, 0) 的直線。包含 *P* 和 *L* 的最小向量空間是什麼？

**17** **(a)** 證明在 M 中，可逆矩陣的集合不是一個子空間。
