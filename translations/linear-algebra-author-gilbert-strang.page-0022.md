## 1.2. 長度和點積

**問題集 1.2**

19

**1** 計算點積 **u** ⋅ **v**、**v** ⋅ **w**、**u** ⋅ (**v** + **w**) 和 **v** ⋅ **w**：

   
   **u** =  [-6, 8] , **v** = [3, 4] , **w** = [8, 6]

**2** 計算向量 **u**、**v** 和 **w** 的長度 ||**u**||、||**v**|| 和 ||**w**||。驗證施瓦茨不等式 **u** ⋅ **v** ≤ ||**u**|| ||**v**|| 和 **v** ⋅ **w** ≤ ||**v**|| ||**w**||。

**3** 在問題 1 中，找到 **v** 和 **w** 的方向上的單位向量，並計算它們之間的角度 θ。選擇向量 **a**、**b**、**c**，使其角度為 0°、90° 和 180°。

**4** 對於任何單位向量 **u** 和 **w**，求出以下點積的實際數值：

   (a) **u** ⋅ **v**  (b) (**u** + **w**) ⋅ (**v** - **w**)  (c) (**w** - 2**u**) ⋅ (**v** + 2**w**)

**5** 找到向量 **u₁** 和 **u₂** 在向量 **v** = (3, 1) 和 **w** = (2, 1, 2) 的方向上。找到單位向量 **U₁** 和 **U₂** 使得它們與 **u₁** 和 **u₂** 垂直。

**6** (a) 描述每個向量 **w** = (w₁, w₂) 垂直於 **v** = (2, -1)。
   (b) 垂直於 **V** = (1, 1, 1) 的向量有哪些？
   (c) 垂直於 **V** = (1, 1, 1) 和 (1, 2, 3) 的向量有哪些？

**7** 找出向量組之間的角度 θ (及其餘弦值)：

   (a) **v** = [1/√3, 1/√3] 和 **w** = [0, 1]
   (b) **v** = [2, 2] 和 **w** = [-1, 2]
   (c) **v** = [1/√3, 1/√3] 和 **w** = [-1, 2]
   (d) **v** = [3, 1] 和 **w** = [-2, 1]

**8** 真或假 (給出一個理由或一個反例)：

   (a) 如果 **u** 垂直於三維空間中的 **v** 和 **w**，那麼向量 **v** 和 **w** 也垂直。
   (b) 如果 **u** 和 **v** 垂直於 **v** 和 **w**，那麼 **u** 垂直於 **v** + 2**w**。
   (c) 如果 **u** 和 **v** 垂直，則單位向量 ||**u**|| = √2。

**9** 箭頭的斜率從 (0, 0) 到 (v₁, v₂) 和從 (0, 0) 到 (w₁, w₂) 是 v₂/v₁ 和 w₂/w₁。假設積 v₂/v₁ * w₂/w₁ = -1。證明箭頭是垂直的。

**10** 從點 (0, 0) 到點 (-1, 2) 和從點 (0, 0) 到點 (-2, 1) 畫出箭頭。將斜率相乘。如果這個乘積是負的，那麼這兩條線是垂直的。

**11** 如果 **u** ⋅ **v** 小於零，則 **u** 和 **v** 之間的角度大於 90°。如果 **u** ⋅ **v** 等於零，則 **u** 和 **v** 垂直。如果 **u** ⋅ **v** 大於零，則 **u** 和 **v** 之間的角度小於 90°。
