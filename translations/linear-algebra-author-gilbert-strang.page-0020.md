## 1.2. 長度和點積 17

輸入 `v` 和 `w` 作為行向量——使用撇號 `'` 將它們轉置為列向量。`2v + 3w` 表示乘以 2 和 3。結果將以列向量的形式呈現，以分號結尾。

**MATLAB** `v = [2 3 4]' ; w = [1 1 1]' ; u = 2 * v + 3 * w`

點積 `v · w` 通常以行向量乘以列向量（不帶點）的形式表示。

代替
```
[2 3]
[4 0]
```
我們更常看到
```
[1 2] [3]
[4 0]
```
或 `v' * w`

向量 `v` 的長度在 MATLAB 中被稱為範數 `norm(v)`。我們可以自己定義為 `sqrt(v' * v)`，使用平方根函數——也稱為平方根。餘弦我們需要自己定義。角度（以弧度為單位）來自反餘弦（acos）函數：

餘弦公式  `cosine = v' * w / (norm(v) * norm(w))`
角度公式  `angle = acos(cosine)`

一個 M-file 將創建一個新的函數 `cosine(v, w)` 以供未來使用。為本書創建的特別的 R 和 PYTHON 開源軟件列表在文末。

## REVIEW OF THE KEY IDEAS

1. 點積 `v · w` 將每個分量相乘：`v₁w₁`，`v₂w₂`，依此類推，並將它們加總：`v₁w₁ + v₂w₂ + ...`
2. 向量 `v` 的長度 `||v||` 是 `v · v` 的平方根。
3. `u = v/||v||` 是一個單位向量。它的長度是 1。
4. 點積 `v · w = 0` 當向量 `v` 和 `w` 垂直時。
5. 兩個非零向量 `v` 和 `w` 之間的角度 `θ` 的餘弦永遠不超過 1：

`cos θ = v · w / (||v|| ||w||)`  Schwarz 不等式  `|v · w| ≤ ||v|| ||w||`

問題 21 將產生三角不等式 `||v + w|| ≤ ||v|| + ||w||`。
