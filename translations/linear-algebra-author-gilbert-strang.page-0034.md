## 第 2 章

**求解線性方程式**

**2.1 向量與線性方程式**

線性代數的核心問題是解一個方程式系統。這些方程式是線性的，這意味著未知數僅被數字相乘——我們從未看到 x 乘以 y。我們的第一個線性系統確實很小。但你會看到它會引導我們到哪裡：

兩個方程式
兩個未知數

```
x - 2y = 1
3x + 2y = 11
```

(1)

我們一次解一列。第一個方程式 x - 2y = 1 產生 xy 平面上的一條直線。點 x = 1, y = 0 在這條線上，因為它解這個方程式。點 x = 3, y = 1 也在這條線上，因為 3 - 2 = 1。如果我們選擇 x = 101，我們會發現 y = 50。

這條直线的斜率是 1/2，因為當 x 改變 2 時，y 增加 1。但斜率在微積分中很重要，這就是線性代數嗎？

![圖 2.1：列圖：線條相交的地方是解。](https://i.imgur.com/a/9q9q9q9.png)

圖 2.1：列圖：線條相交的地方是解。

圖 2.1 顯示線 x - 2y = 1。第二條線在這個“列圖”中來自第二個方程式 3x + 2y = 11。你不能錯過線條相交的點，那裡是解。
