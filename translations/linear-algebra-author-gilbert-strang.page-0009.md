這是典型的情況！直線、然後平面、然後空間。但還有其他可能性。當 **w** 恰好是 c**u** + d**v** 時，第三個向量位於前兩個向量所構成的平面上。**u**、**v**、**w** 的組合將不會超出該 **uv** 平面。我們無法得到完整的三維空間。請思考問題 1 中的特殊情況。

■ 關鍵概念回顧 ■

1.  二維空間中的向量 **v** 有兩個分量 **v**₁ 和 **v**₂。
2.  **v** + **w** = (**v**₁ + **w**₁, **v**₂ + **w**₂) 和 c**v** = (c**v**₁, c**v**₂) 是一次一個分量地計算出來的。
3.  三個向量 **u**、**v** 和 **w** 的線性組合是 c**u** + d**v** + e**w**。
4.  取 **u** 的*所有*線性組合，或 **u** 和 **v** 的所有組合，或 **u**、**v**、**w** 的所有組合。在三維空間中，這些組合通常會填滿一條直線，然後是一個平面，以及整個空間 **R**³。

■ 範例 ■

**1.1 A** 向量 **v** = (1, 1, 0) 和 **w** = (0, 1, 1) 的線性組合構成一個平面。*描述*這個平面。找出一個*不是* **v** 和 **w** 組合的向量。

**解答** 組合 c**v** + d**w** 在 **R**³ 中構成一個平面。該平面中的向量允許任何 c 和 d 值。圖 1.3 中的平面填補了「**u**-線」和「**v**-線」之間。

組合 c**v** + d**w** = c \[ 1 1 0 ] + d \[ 0 1 1 ] = \[ c c+d d ] 構成一個平面。

該平面中的四個特定向量是 (0, 0, 0)、(2, 3, 1)、(5, 7, 2) 和 (π, 2π, π)。第二個分量 c + d 始終是第一個和第三個分量之和。向量 (1, 2, 3) *不在*該平面中，因為 2 ≠ 1 + 3。
