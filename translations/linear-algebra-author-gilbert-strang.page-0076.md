## 2.4. 矩陣運算規則 73

HTH 和 THH。這些是「我選擇 j」的例子，即 L 中行 j 和列 i 的數量。如果我們從 0 開始計數行和列，則 i = 0 且 j = 0 (並記住 0! = 1):

```
cᵢⱼ = ( i ) = i choose j = i!
     ( j )           j!(i-j)!
```

有六種方式選擇四個中的兩個 aces。我們將使用帕斯卡三角形和這些矩陣。以下是我想問你的問題：

1.  H 是 L²？這是「超矩陣」。
2.  將 H 乘以向量和冪。
3.  最後一行是 H 的 8, 12, 6, 1。一個立方體有 8 個頂點，12 個邊，6 個面，1 個盒子。
    H 的下一個會如何建構一個 4D 超立方體？

**解答**

將 L 乘以 L 得到超矩陣 H = L²：

```
[ 1  1 ]   [ 1  1 ]   [ 1  1 ]   [ 2  1 ]
[ 1  2 ] * [ 1  2 ] = [ 2  4 ] = [ 4  4 ] = H
[ 1  3 ]   [ 1  3 ]   [ 8  12 ]   [ 8  12 ]
```

現在將 H 乘以向量和冪：

```
[ 1  1 ]   [ 1 ]   [ 1 ]   [ 1 ]
[ 2  4 ] * [ x ] = [ 3 ] = [ x ] * [ 2 + x ]
[ 4  4 ]   [ x²]   [ 9 ]   [ x² ] * [ (2 + x)² ]
[ 8  12 ]   [ x³]   [ 27]   [ x³ ] * [ (2 + x)³ ]
```

如果 x = 1，我們得到 1 + x 的冪。如果 x = 0，我們得到 2 的冪。當 L 產生 1 + x 的冪時，L 再次產生 2 + x 的冪。

**如何計算立方體的頂點、邊、面和方塊的數量？**一個正方形在 2D 中有 4 個頂點，4 個邊。在 1 次迭代中，它會增加一個邊和一個面。

連接兩個正方形以獲得 3D 立方體。連接三個立方體以獲得 4D 超立方體。

立方體有 8 個頂點和 12 個邊；4 個邊在每個正方形中，4 個邊在兩個正方形之間。
它有 6 個面；在每個正方形的 4 個面上，每個正方形都有 2 個面。
它有 1 個盒子。
我們可以從下一行 8, 12, 6, 1 開始。
這行是 2⁴；hᵢ₊₁ = hᵢ + hᵢ₋₁。
這行是 16, 24, 12, 4。
為了得到這個結果，我們需要 2 個立方體。
為了得到兩個立方體，我們需要 4 個正方形。
為了得到 4 個正方形，我們需要 8 個正方形。
為了得到 8 個正方形，我們需要 16 個正方形。
為了得到 16 個正方形，我們需要 32 個正方形。
因此，h₀ = 1, h₁ = 1, h₂ = 2, h₃ = 3, h₄ = 5, h₅ = 8, h₆ = 13。
