## 56 第 2 章：解線性方程式

### 2.3 使用矩陣進行消去法

我們現在將結合兩個想法——消去法和矩陣。目標是以最清晰的方式表達消去法的所有步驟。對於 3x3 的例子，消去法可以描述為從某行減去另一行的倍數——對於更大的系統，長列表的步驟將會毫無希望。你將看到如何使用矩陣 *E* 從第 *i* 行減去第 *j* 行的倍數。

前一節的 3x3 例子可以優美地寫成簡短形式 *Ax = b*：

```
2x₁ + 4x₂ - 2x₃ = 2
4x₁ + 9x₂ - 3x₃ = 8   是相同的   [ 2  4 -2 ] [ x₁ ] = [ 2 ]  (1)
-2x₁ - 3x₂ + 7x₃ = 10                      [ 4  9 -3 ] [ x₂ ] = [ 8 ]
                                             [-2 -3  7 ] [ x₃ ] = [ 10 ]
```

這些數字進入矩陣 *A*。這個矩陣不僅坐在 *x* 的旁邊，它還乘以 *x*。規則是 “*A* 乘以 *x*” 精確地選擇產生三個方程式。

**矩陣 *A* 的回顧。** 矩陣乘以一個向量產生一個向量。矩陣是正方形的，因為方程式的數量（三個）與未知數的數量（三個）相符。我們的矩陣是 3x3。一個一般正方形矩陣是 *n* x *n*。然後向量 *x* 是 *n* 維向量。

**在 R³ 中的未知數**  *x* =  [ x₁ ]  且解是 *x* = [ 2 ]
                                  [ x₂ ]                               [ 1 ]
                                  [ x₃ ]                               [ 2 ]

關鍵點：*Ax = b* 代表方程式的行形式和列形式。

**列形式** *Ax* = (-1) [ 2 ] + 2 [ 4 ] + 3 [ -2 ] = [ 2 ]
                         [ 4 ]   [ 9 ]   [ -3 ]   [ 8 ]
                         [-2 ]   [-3 ]   [ 7 ]   [ 10 ]
                         = *b*。

這個 *Ax* 的規則被用得如此頻繁，以至於我們再次強調它。

*Ax* 是 *A* 的列的組合。*A* 的成分乘以 *x* 乘以那些列：

*Ax* = *x₁* 乘以 (第 1 列) + *x₂* 乘以 (第 2 列) + ... + *xₙ* 乘以 (第 *n* 列)。

當我們計算 *Ax* 的成分時，我們使用行向量的點積（矩陣乘法的一種形式）。第 *i* 個成分是 *A* 的第 *i* 行和 *x* 的點積，記為 *aᵢ₁x₁ + aᵢ₂x₂ + ... + aᵢₙxₙ*。

**Ax 的第 *i* 個成分** 是 *aᵢ₁x₁ + aᵢ₂x₂ + ... + aᵢₙxₙ*。記為 ∑ *aᵢⱼxⱼ*。

**Sigma 符號 ∑** 是一個指令，表示 “加總”。*j* = 1 到 *n* 的加總。它只是簡潔的符號。

**舉例：**  ∑ᵢ *aᵢⱼ* 表示所有 *aᵢⱼ* 的加總，其中 *j* 是固定的，而 *i* 從 1 到 *n*。*Einstein* 慣例（或 *Einstein* 求和慣例）是指當一個指標在一個項中出現兩次時，就對該指標進行求和。
