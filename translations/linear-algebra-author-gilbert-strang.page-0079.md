        [ 0  1 ]

**9** 將 A 的第一行加到第二行。這會產生 EA 以下。然後將 EA 的第一列加到第二列以產生 E(A)F：

   EA =  [ 1  0 ] [ a  b ] = [ a  b ]
         [ 1  0 ] [ c  d ] = [ a+c  b+d ]

   和 (EA)F = (EA) [ 0  1 ] = [ a  b ]
                   [ 0  1 ]   [ a+c  a+b ]
                               [ c  d ]

   (a) 以相反的順序執行這些步驟。首先將 A 的第一列加到 F 的第二列，然後將 A 的第一行加到 F 的第二行以產生 E(AF)。
   (b) 比較 (EA)F 和 E(AF)。什麼法則支配矩陣乘法？

**10** 將 A 的第一行加到第二行以產生 E4。然後將 F 加到 E4 的第二行以產生 F(EA)：

   F(EA) = [ 1  1 ] [ a+c  b+d ] = [ 2a+c  2b+d ]
           [ 0  1 ] [ a+c  b+d ] = [ a+c  b+d ]

   (a) 這些步驟的順序是否被矩陣乘法法則所支配？首先將 A 的第一行加到第二行，然後將 F 加到 E4 的第二行。
   (b) 什麼法則支配或不支配矩陣乘法？
