## 68 第 2 章：解線性方程式

```
a₁₁ a₁₂ … a₁₅   *   b₁j   *   *   (AB)₁j   *   *   *
*   b₂j   *   *   (AB)₂j   *   *
*   …   *   *   …   *   *
*   b₅j   *   *   (AB)₅j   *   *
```

A 是 4 x 5   B 是 5 x 6   AB 是 4 x 6

圖 2.8：這裡 i = 2 且 j = 3。則 (AB)₂₃ = (row 2) • (column 3) = Σa₂kb₃k。

**範例 1**  方陣才能相乘，且只有在它們具有相同大小的情況下：

```
[ 1  1 ] [ 2  5 ] [ 6  1 ]
[ 2 -1 ] [ 3  4 ] [ 0  0 ]
```

第一個點積是 1 • 2 + 1 • 3 = 5。三個點積產生 6, 1, 和 0。每個點積需要兩個乘法運算。因此，AB 包含 n² 個點積，row A 乘以 column B。每個點積需要 n 次乘法運算，因此 AB 的計算需要 n³ 次乘法運算。對於 n = 100，我們需要一千億次乘法運算。對於 n = 2，我們需要 n³ = 8。

數學家們原本認為要計算 AB 絕對需要 2³ = 8 次乘法運算。但有人發現了方法，可以將其減少到 7 次（和額外加法運算）。透過將 n 個 n x n 矩陣分成 2 x 2 的區塊，這個想法減少了矩陣乘法的數量，從 n³ 降至 n².8，而指數持續下降！目前最佳的數值是 n²․376。但演算法太笨拙，科學計算通常使用常規方法：n² 次點積來計算 AB，以及 n² 次乘法運算。

**範例 2**  假設 A 是一個 row vector (1 x 3) 且 B 是一個 column vector (3 x 1)。則 AB 是 1 x 1 (僅有一個 entry，點積)。另一方面，B 乘以 A (column 乘以 row) 是一個 3 x 3 矩陣。此乘法運算允許：

```
Column times row   [ 0  0  0 ]
(n x 1) (1 x n) = (n x n)   [ 1  1  2 ]
                               [ 2  2  6 ]
```

一個 row times a column 是一個 “inner” product—這是一種點積的另一種名稱。一個 column times a row 是一個 “outer” product。這是一種矩陣乘法的另一種方式。

**Rows and Columns in A**

在最後一個例子中，A 的 column space 是由 B 產生的。換句話說，B 位於 A 的 column space 中。我們將在第 3 章中更詳細地研究 column space 和 row space。
