## 3.2. A 的零空間：解 Ax = 0

**135**

**1. 前向消去將 A 轉換為一個三角矩陣 U (或其簡化行階梯形式)。**

**2. 反向代入在 Ux = 0 或 Rx = 0 中產生 x。**

你會注意到在反向代入時，當 A 和 U 的樞軸數少於 *n* 時會出現差異。我們在本章允許所有矩陣（不僅僅是方陣，包括可逆矩陣）。

樞軸仍然是非零的。樞軸下方的欄位仍然可以是零。但一個欄位可能沒有樞軸。如果一個欄位沒有樞軸，那麼該欄位就不會影響計算。繼續到下一個欄位。第一個例子是一個 3x4 矩陣，有兩個樞軸：

```
A =  [ 1  1  2  3 ]
     [ 2  2  8 10 ]
     [ 3 10 13 ]
```

顯然 a₁₁ = 1 是第一個樞軸。清除第二欄和第三欄下方的樞軸：

```
[ 1  1  2  3 ]  ->  [ 1  1  2  3 ]
[ 0  0  4  4 ]      [ 0  0  4  4 ]  (減去 2 倍第一列)
[ 0  4  4 ]          [ 0  4  4 ]  (減去 3 倍第一列)
```

第二欄包含一個零在樞軸位置。我們尋找零在非零項中的位置。進入該欄位的那個位置是零。消去不能對第二欄做任何事情。這表示問題，這表示我們期望在一個矩陣中看到。沒有理由要將其歸零，並且將其歸零到第三欄。

第二個樞軸是 4 (在第三欄)。從第二欄減去 2 倍第三欄清除該欄位下方的樞軸。**樞軸欄位是 1 和 3。**

```
Triangular U:  U = [ 1  1  2  3 ]
                   [ 0  0  4  4 ]
                   [ 0  0  0  0 ]
```

只有兩個樞軸
最後一個方程式
變成 0 = 0

第四欄也包含一個零在樞軸位置——但不能再做任何事情。矩陣現在是完整的。矩陣有三行，四個欄位，只有兩個樞軸。原始 Ax = 0 似乎要解三個不同的方程式，但第三個方程式是前兩個方程式的線性組合。消去揭示了內在的真相，關於方程式系統的資訊。現在我們必須從 U x = 0 求解。這揭示了自由變數和唯一解。

現在我們做反向代入，以尋找所有解。

A. 為了避免混淆，我們將變數寫成 x₁, x₂, x₃ 和 x₄。

B.  樞軸變數是 x₁ 和 x₃。自由變數是 x₂ 和 x₄。它們可以任意取值。
