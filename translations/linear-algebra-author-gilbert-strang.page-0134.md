## 3.1. 向量空間 131

**挑戰問題**

**30** 假設 S 和 T 是向量空間 V 的兩個子空間。

(a) 定義：S + T 包含所有向量 s + t，其中 s 來自 S，t 來自 T。證明 S + T 滿足向量空間的要求（加法和純量乘法）。

(b) 如果 S 和 T 是 R<sup>m</sup> 中的線，S + T 和 S ∪ T 有什麼區別？該聯集包含來自 S 或 T 或兩者的所有向量。解釋此陳述：S ∪ T 的張成是 S + T。（第 3.5 節回顧“張成”一詞。）

**31** 如果 S 是 A 的列空間，T 是 C(B)，則 S + T 是什麼矩陣 M 的列空間？A 和 B 的列都是在 R<sup>n</sup> 中。（我不認為 A + B 總是正確的矩陣。）

**32** 證明矩陣 A 和 [A AB]（具有額外列的矩陣）具有相同的列空間。但找到一個小於 C(A) 的平方矩陣，使得 C(A<sup>2</sup>) 更小。重要提示：一個 n × n 矩陣 C(A) = R<sup>n</sup> 正確地當 A 是一個 ______ 矩陣時。
