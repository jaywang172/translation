## 第 61 頁

**矩陣乘法**

我們已經知道如何預期 *EA* 的結果。此乘法器 *l* 減去 2 倍的 *E* 的第一列，得到 *A* 的第二列。乘法器是 *l* = -2。

```
EA =  [ 1  0  0 ] [ 1  0  2 ]   [ 1  0  -2 ]
      [-2  1  0 ] [ 0  1  4 ] = [ -2  1  -4 ]  (帶有零)
      [ 0  0  1 ] [-1  -3  1 ]   [ 0  0  1 ]
```

此步驟並未改變 *A* 的第一列和第三列。*A* 的第二列已改變。只有矩陣乘法符合消除的規則——此結果與我們期望的相同。矩陣 *EA* 等於從 *A* 減去 2 倍的 *A* 的第一列的乘積，即 *EA* = *A* - 2*l* *v*<sup>T</sup>，其中 *v*<sup>T</sup> 是 *A* 的第一列。

現在我們看看 *Ax* = *b*。我們有 *E* *Ax* = *Eb*。左側是 *EAx*，因此 *EAx* = *Eb*。這就是我們想要得到的。*E* 作用於 *A*，作用於 *x*，作用於 *b*。

重要的是，*E* 作用於 *A* 的方式，作用於 *x* 的方式，作用於 *b* 的方式是相同的。*E* 作用於 *A*，產生 *EA*。*E* 作用於 *x*，產生 *Ex*。*E* 作用於 *b*，產生 *Eb*。

**矩陣乘法的規則**

如果 *A* 是 *m* × *n* 矩陣，*B* 是 *n* × *p* 矩陣，則 *AB* 是 *m* × *p* 矩陣。

為了得到 *AB* 的 (i, j) 元素，我們將 *A* 的第 *i* 列與 *B* 的第 *j* 行進行點積：

```
(AB)<sub>ij</sub> = A<sub>i1</sub>B<sub>1j</sub> + A<sub>i2</sub>B<sub>2j</sub> + ... + A<sub>in</sub>B<sub>nj</sub>
```

這表示 *AB* 的每一列是 *A* 的列的線性組合，係數來自 *B* 的對應行。

**重要提示：** *AB* 和 *BA* 通常不同。矩陣乘法不滿足交換律。
