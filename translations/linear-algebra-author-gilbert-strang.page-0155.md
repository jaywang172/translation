## 152 章節 3. 向量與子空間

**2** 求下列各矩陣的列簡化梯形式 *R* 與秩：

(a) 3x4 矩陣，所有元素皆為 4。
(b) 3x4 矩陣，其中 *a<sub>ij</sub>* = *i + j - 1*。
(c) 3x4 矩陣，其中 *a<sub>ij</sub>* = (-1)<sup>*i+j*</sup>。

**3** 求下列各（區塊）矩陣的列簡化梯形式 *R*：

```
A = [ 0 0 0 ]
    [ 0 0 3 ]
    [ 2 4 6 ]

B = [ A  A ]
    [ A  A ]

C = [ A  A ]
    [ 0  0 ]
```

**4** 假設樞軸變數先被消去。描述在列簡化梯形式中（區塊 *B* 應為 *r* by *r*）：

```
R = [ A  B ]
    [ C  D ]
```

**5** (愚蠢問題) 描述兩個 2x3 矩陣 *A<sub>1</sub>* 和 *A<sub>2</sub>*，它們的列簡化形式分別為 *R<sub>1</sub>* 和 *R<sub>2</sub>*，使得 *R<sub>1</sub> + R<sub>2</sub>* 是列簡化形式的 *A<sub>1</sub> + A<sub>2</sub>*。是否 *r<sub>1</sub> = A<sub>1</sub>* 且 *r<sub>2</sub> = A<sub>2</sub>* 在這種情況下成立？ *R<sub>1</sub> - R<sub>2</sub>* 是否等於 *ref(A<sub>1</sub> - A<sub>2</sub>)*？

**6** 如果 *A* 有 *r* 個樞軸列，那麼 *A<sup>T</sup>* 有 *r* 個樞軸列嗎？給出一個 3x3 範例，其中樞軸列的列號在 *A* 和 *A<sup>T</sup>* 中不同。

**7** 對於這些 *R*，求特殊解 *Rx = 0* 和 *y<sup>T</sup>R = 0*：

```
R = [ 1 0 2 ]
    [ 0 1 3 ]
    [ 0 0 0 ]

R = [ 0 1 2 ]
    [ 0 0 0 ]
    [ 0 0 0 ]
```
