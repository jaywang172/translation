## 167

**第 3 章 線性代數與子空間**

**練習 3.2**

1.  解下列線性系統：
    (a)  
    ```
    x₁ - x₂ + 2x₃ = 1
    2x₁ - 2x₂ + 4x₃ = 2
    x₁ - x₂ + 2x₃ = 1
    ```
    (b)
    ```
    x₁ + x₂ + x₃ = 1
    2x₁ + 2x₂ + 2x₃ = 2
    x₁ + x₂ + x₃ = 1
    ```

2.  對於下列矩陣 A，求出 A 的零空間。
    (a)
    ```
    A = [ 1  2  3 ]
        [ 2  4  6 ]
    ```
    (b)
    ```
    A = [ 1  0  -1 ]
        [ 0  1  1 ]
        [ 1  1  0 ]
    ```

3.  對於下列矩陣 A，求出 A 的列空間。
    (a)
    ```
    A = [ 1  2  3 ]
        [ 2  4  6 ]
    ```
    (b)
    ```
    A = [ 1  0  -1 ]
        [ 0  1  1 ]
        [ 1  1  0 ]
    ```

4.  若 A 是 m × n 矩陣，且 A 的零空間只包含零向量，則 A 的列是線性獨立的嗎？為什麼？

5.  若 A 是 m × n 矩陣，且 A 的列空間是 ℝᵐ，則 A 的行是線性獨立的嗎？為什麼？

6.  若 A 是 m × n 矩陣，且 A 的零空間是 ℝⁿ，則 A 的列空間是什麼？

7.  若 A 是 m × n 矩陣，且 A 的列空間是 ℝⁿ，則 A 的行空間是什麼？

8.  若 A 是 m × n 矩陣，且 A 的行空間是 ℝᵐ，則 A 的零空間是什麼？

9.  若 A 是 m × n 矩陣，且 A 的列空間與 A 的行空間相同，則 A 是什麼形狀的矩陣？

10. 考慮下列矩陣：
    ```
    A = [ 1  2  1 ]
        [ 2  4  2 ]
        [ 3  6  3 ]
    ```
    (a) 求出 A 的零空間。
    (b) 求出 A 的列空間。
    (c) A 的行空間是什麼？

11. 考慮下列矩陣：
    ```
    A = [ 1  0  1 ]
        [ 0  1  1 ]
        [ 1  1  2 ]
    ```
    (a) 求出 A 的零空間。
    (b) 求出 A 的列空間。
    (c) A 的行空間是什麼？

12. 假設 A 是 5 × 7 矩陣，且 A 的零空間的維度是 3。求出 A 的列空間的維度。

13. 假設 A 是 8 × 5 矩陣，且 A 的列空間的維度是 4。求出 A 的零空間的維度。

14. 假設 A 是 m × n 矩陣，且 A 的零空間的維度是 n - r，其中 r 是 A 的秩。求出 A 的列空間的維度。

15. 假設 A 是 m × n 矩陣，且 A 的列空間的維度是 r。求出 A 的零空間的維度。

16. 說明為什麼 A 的零空間與 A 的左零空間（Aᵀ 的零空間）是正交補空間。

17. 假設 A 是 m × n 矩陣。證明 A 的列空間與 A 的行空間的維度之和等於 n。

18. 假設 A 是 m × n 矩陣。證明 A 的列空間與 A 的零空間的維度之和等於 n。

19. 假設 A 是 m × n 矩陣。證明 A 的行空間與 A 的零空間的維度之和等於 m。

20. 假設 A 是 m × n 矩陣。證明 A 的行空間與 A 的左零空間的維度之和等於 m。
