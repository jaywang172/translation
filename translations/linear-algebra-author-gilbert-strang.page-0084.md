## 2.5 反矩陣

假設 *A* 是一個方陣。我們尋找一個與 *A* 相同大小的“反矩陣” *A*<sup>-1</sup>，使得 *A*<sup>-1</sup> 乘以 *A* 等於 *I*。無論 *A* 做什麼，*A*<sup>-1</sup> 都會把它撤銷。它們的乘積是單位矩陣——它什麼也不做向量，所以 *A*<sup>-1</sup>*A* = *x*。但 *A*<sup>-1</sup>*x* 可能不等於 *x*。

什麼是矩陣主要做的事情？它乘以一個向量 *x*。將 *Ax* = *b* 乘以 *A*<sup>-1</sup> 得到 *A*<sup>-1</sup>*Ax* = *A*<sup>-1</sup>*b*。這就是 *x* = *A*<sup>-1</sup>*b*。乘積 *A*<sup>-1</sup>*A* 就像乘以一個數，然後除以該數。一個數的反數（如果該數不為零）更複雜，更感興趣。矩陣 *A*<sup>-1</sup> 被稱為“A 的反矩陣”。

**定義** 矩陣 *A* 是可逆的，如果存在一個矩陣 *A*<sup>-1</sup> 使得：

*A*<sup>-1</sup>*A* = *I*  且  *AA*<sup>-1</sup> = *I*.  (1)

並非所有矩陣都有反矩陣。這就是我們首先要問的問題：*A* 是否可逆？我們不一定需要立即計算 *A*<sup>-1</sup>。在大多數問題中，我們不需要計算它！以下是關於 *A*<sup>-1</sup> 的六個“注意事項”。

**Note 1** 反矩陣存在當且僅當消去法產生主元（行交換是被允許的）。消去法解 *Ax* = *b* 而不涉及除以零時，需要 *A*<sup>-1</sup>。

**Note 2** 矩陣 *A* 不能有兩個不同的反矩陣。假設 *BA* = *I* 且 *AC* = *I*。那麼 *B* = *C*，根據這個“證明”：

*B*(*AC*) = (*BA*) *C*  給出  *BI* = *IC*  或  *B* = *C*.  (2)

這表明左反矩陣 *B*（從左邊乘以）和右反矩陣 *C*（從右邊乘以）必須是相同的矩陣。

**Note 3** 如果 *A* 是可逆的，那麼 *Ax* = *b* 的唯一解是 *x* = *A*<sup>-1</sup>*b*。

將 *Ax* = *b* 乘以 *A*<sup>-1</sup>。那麼 *x* = *A*<sup>-1</sup>*Ax* = *A*<sup>-1</sup>*b*。

**Note 4** (重要) 假設存在一個非零向量 *x* 使得 *Ax* = 0。那麼 *A* 不能有反矩陣。矩陣 *A* 無法將 0 映射回任何非零向量。

如果 *A* 是可逆的，那麼 *A*<sup>-1</sup>*0 = 0 只能有零解 *x* = 0。

**Note 5** 2 x 2 矩陣是可逆的當且僅當 *ad - bc* 不為零：

2 by 2 Invertible:
*A* =  
[ a  b ]
[ c  d ]
如果 *ad - bc* ≠ 0.  (3)

這個數字 *ad - bc* 被稱為矩陣 *A* 的行列式，記作 det(*A*)。如果行列式為零，矩陣是奇異的。
