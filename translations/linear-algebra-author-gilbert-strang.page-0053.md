## 50 章 2. 解線性方程式

**■ 關鍵概念回顧 ■**

1. 線性系統 (Ax = b) 在消去法後會變成三角矩陣 (Ux = c)。
2. 我們從方程式 i 中減去方程式 j 的 *l<sub>ij</sub>* 倍，以使 (i, j) 項為零。主軸位置不能為零！
3. 乘數 *l<sub>ij</sub>* =  主軸位置的元素 / 第 j 列主軸位置的元素。
4. 主軸位置的零可以透過列交換來修復（如果它是非零元素）。
5. 上三角矩陣系統透過回代法求解（從底部開始）。
6. 如果崩潰是永久性的，則該系統無解或有無限多個解。

**■ 運算範例 ■**

2.2 A 當消去法應用到此矩陣 A 時，什麼是第一個和第二個主軸位置？在第一次步驟中，*l<sub>21</sub>* 乘以第 1 列並從第 2 列中減去。

A 具有第一列和第二列的差為 -1, 2, -1。

```
A =  [ -1  1  0 ]      [ 1  0  0 ]
     [ -1  2 -1 ]  ->  [ 0  1 -1 ]
     [ 0 -1  2 ]      [ 0  0  2 ]
```

-> U =

什麼元素在 2,2 位置（而不是 0）會迫使第 2 列和第 3 列交換？為什麼較低的乘數 *l<sub>31</sub>* = 0，需要從第 1 列減去 2 倍才能從第 3 列中減去？
如果你將角落元素 *a<sub>33</sub>* 從 2 更改為 1，為什麼消去法會失敗？

**解答** 第一個主軸位置是 1。乘數 *l<sub>21</sub>* = -1/-1 = 1。當從第 2 列減去第 1 列時，第二個主軸位置顯示為 1。

如果我們將中間元素從“2”更改為“1”，這將迫使交換列。（零將出現在第二個主軸位置。）乘數 *l<sub>31</sub>* 是零，因為 *a<sub>31</sub>* = 0。零不需要消去。

最後的樞軸是 1。如果原角落元素 *a<sub>33</sub>* 減少為 1（如 *a<sub>33</sub>* = 1），消去法將會失敗。沒有第三列，消去法失敗。

2.2 B 假設 A 已經是三角矩陣（上三角或下三角）。那麼 A 是否總是具有唯一解？

**解答** 三角矩陣的樞軸位置總是為非零。消去法不會改變主軸位置。因此，三角矩陣總是具有唯一解，除非主軸位置為零。在這種情況下，該系統具有無限多個解或無解。
