## 2.7. 轉置與置換 113

在 MATLAB 中，A(k, :) = A(k, :) 交換第 k 列與第 r 列以下 (其中第 k 樞元已找到)。然後 lu code 更新 L 與 P，以及 P 的符號：

```
A(r, k, :) = A(k, r, :)
L(r, k, 1 : k - 1) = L(k, r, 1 : k - 1)
P(r, k, :) = P(k, r, :)
sign = -sign
```

[L, U, P] = lu(A)

“P 的 ‘符號’” 表示行交換的次數是偶數 (符號 = +1)。奇數次的行交換會產生符號 = -1。在開始時，P = I，且符號 = +1。當 P 進行行交換時，符號會反轉。符號的最終值是行列式 det(P) 與 det(A) 的乘積，且不取決於行交換的順序。

對於 PA，我們無法回到熟悉的 LU。這通常是標準分解。實際上，lu(A) 讓我們回到第一次可用樞元。數學上，我們可以接受一個小樞元，任何比零大的值。但如果計算機向下選擇列中最大的值 (第 9.1 節解釋了為什麼“部分樞元”可以減少捨入誤差)。因此 P 可能包含並非代數上必要的交換。但 P 仍然是 LU。

我們的目標是理解置換矩陣所必需的計算。計算 A = LU 很容易做到手算，而不需要計算機的幫助。Teaching Code splu(A) 因子分解 PA = LU，且 splv(A, b) 解 Ax = b 對任何不可逆的 A。程式停止如果未在列中找到樞元。那麼 A 是不可逆的。

**■ REVIEW OF THE KEY IDEAS ■**

1. 轉置將 A 的行轉換為 A 的列。因此 (Aᵀ)ᵢⱼ = Aⱼᵢ。

2. AB 的轉置是 BᵀAᵀ。A⁻¹ 的轉置是 (Aᵀ)⁻¹。

3. 點積是 x ⋅ y = xᵀy。Then (Ax)ᵀy 等於 xᵀ(Aᵀy)。

4. 當 A 是對稱矩陣 (A = Aᵀ) 時，其 LDU 分解是對稱的：A = LDLᵀ。

5. 一個置換矩陣 P 具有 1 在每一行和每一列，且 Pᵀ = P⁻¹。

6. 有 n! 個置換矩陣，大小為 n。一半是偶數，一半是奇數。

7. 如果 A 具有不為零的樞元，則 PA = LU。
