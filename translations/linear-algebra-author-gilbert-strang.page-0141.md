## 138 章節 3. 向量空間與子空間

一個短而寬的矩陣 (n > m) 總是具有非零的零空間。必須至少有 n - m 個自由變數，因為向量的數量不能超過 m。 (矩陣的秩只有 n 行，而一個行向量最多只有兩個 pivots。) 當然，一個矩陣可能沒有 pivot，這意味著存在自由變數。但這裡的重點是：當存在一個自由變數時，它可以被設定為 1。然後方程式 Ax = 0 會有一個非零解。

重複：最多有 m 個 pivots。當 n > m 時，系統 Ax = 0 會有一個非零解。實際上，有無限多個解，因為任何 Ax = 0 的倍數 cx 也是一個解。零空間包含一條線的解。當有兩個自由變數時，零空間是特殊的解，且零空間是偶數維的。

零空間是一個子空間。它的“維度”是自由變數的數量。這個核心概念——子空間的維度——在本章中定義並解釋。

### 縮減列階梯矩陣

從一個階梯矩陣 U 我們再往前一步。繼續使用一個 3x4 的例子：

```
U =  [ 1  2  3 ]
     [ 0  0  4 ]
     [ 0  0  0 ]
```

我們可以將第二行除以 4。然後兩個 pivots 都等於 1。我們可以從第一行減去 2 倍的第二行。縮減的列階梯矩陣 R 如下所示：

```
縮減列階梯矩陣   R = rref(A) = [ 1  0  1 ]
                     [ 0  0  1 ]
                     [ 0  0  0 ]
```

Pivot 行
包含 1

R 具有 1s 作為 pivots。零以上的所有數字都來自向上消去。

**重要：** 如果 A 是可逆的，則縮減的列階梯形式是單位矩陣 R = I。這是最終的列階梯形式。當然，零空間是 {0}。

R 中的零表示可以找到與之相同的解 (如前所述)。

1.  設 x₂ = 1 且 x₄ = 0。解得 Rx = 0。則 x₁ = -1 且 x₃ = 0。
    這些數字 -1 和 0 坐落在第二列 (R 中的正號)。

2.  設 x₂ = 0 且 x₄ = 1。解得 Rx = 0。則 x₁ = -1 且 x₃ = -1。
    這些數字 -1 和 -1 坐落在第二列 (R 中的正號)。

通過反轉符號，我們可以讀取零空間的基底向量 (直接從 R 中)。零空間的基底向量是 N = [ -1  0  0  1 ] 和 N = [ -1  1  0  0 ]。

```
N = [ -1 ]   N = [ -1 ]
    [  0 ]   [  1 ]
    [  0 ]   [  0 ]
    [  1 ]   [  0 ]
```

零空間的維度是 2 (R 中的自由變數的數量)。

在上述例子中，A 的列空間由第一和第三列 A 的線性組合組成。因此，列空間的基底向量是 [ 1  0  0 ] 和 [ 3  4  0 ]。列空間的維度是 2 (A 中的 pivots 的數量)。
