## 26 章節 1. 向量導論

每個常數向量 (c, c, c) 具有循環差異的積分。這個未確定常數 c 類似於在積分中我們添加的常數。循環差異在第一個成分中，從 x₀ = 0 開始。
另一種可能性是 Cx = b 沒有解：

```
Cx = b   | x₁ - x₃ |   | 1 |   左側必須為 0
          | x₂ - x₁ | = | 3 |   右側添加至 x₂, x₃
          | x₃ - x₂ |   | 5 |   無解 x₁, x₂, x₃
```
(13)

從幾何角度來看，沒有 u, v, 和 w* 的任何組合可以產生向量 b = (1, 3, 5)。這些組合無法填滿整個三維空間。右側必須有 b₁ + b₂ + b₃ = 0 才能有 Cx = b 的解，因為左側 x₁ - x₃, x₂ - x₁, 和 x₃ - x₂ 總是加總為零。

換句話說，所有線性組合 x₁u + x₂v + x₃w* 都在平面 b₁ + b₂ + b₃ = 0 上。這是一個將代數與幾何聯繫起來的主題。線性組合可以填滿所有空間，或僅僅是一個平面。我們需要一張圖來顯示 u, v, w* (第一個例子) 和 u, v, w* 的關鍵區別。

**獨立性與相依性**

圖 1.10 顯示了矩陣 A 和 C 的列向量。前兩列在兩個圖中都是相同的。如果我們只考慮這兩個向量，那麼它們的組合將會得到一個二維平面。關鍵問題是第三個向量是否在這個平面上。

*   **獨立性** w* 不在 u 和 v 的平面上。
*   **相依性** w* 在 u 和 v 的平面上。

重要的是，新的向量 w* 可以表示為 u 和 v 的線性組合：

```
u + v + w* = 0   =>   w* =  | -1 | = -u - v.
                      |  0 |
                      |  1 |
```
