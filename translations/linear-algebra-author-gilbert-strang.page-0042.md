## 2.1 向量與線性方程式 39

**工作範例**

**2.1 A** 描述下列方程式組的行向量圖：Ax = b。透過仔細檢查 A 的行向量（避免使用消去法）。

```
x + 3y + 2z = -3
2x + 2y + 2z = -2
3x + 5y + 4z = -5
```

這可以寫成：

```
[ 1  3  2 ] [ x ] = [ -3 ]
[ 2  2  2 ] [ y ] = [ -2 ]
[ 3  5  4 ] [ z ] = [ -5 ]
```

**解：** 行向量圖要求一個線性組合，這個線性組合的結果是 **b**，且來自 A 的三個行向量。由於 **b** 是第二個行向量的倍數，因此解為 x = 0, y = -1, z = 0。要證明 (0, -1, 0) 是唯一的解，我們需要知道 “A 是可逆的” 且 “A 的行向量是獨立的” 以及 “行列式不為零”。

那些文字並未定義由消去法測試的條件，且（對於這個矩陣）我們需要一個由三個非零元素組成的滿秩。

假設右側變更為 **b** = (4, 4, 8)。這個好組合的結果是 x = 1, y = 1, z = 0。

**2.1 B** 這個系統沒有解。這三個平面不相交於一點。沒有任何三個行向量的線性組合可以產生 **b**。如何證明這一點？
