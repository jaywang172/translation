我們可以加總 **R**ⁿ 中的任何向量，並且可以將任何向量乘以任何純量。

「在向量空間中」意味著結果仍然在空間中。如果 **v** 是 **R**⁴ 中的向量 (1, 0, 0, 0)，那麼 2**v** 是 **R**⁴ 中的向量 (2, 0, 0, 0)。(在這種情況下，純量 1) 一個完整的性質集可以在 **R**ⁿ 中找到。交換律 **v** + **w** = **w** + **v**；分配律 **c**(**v** + **w**) = **c**v + **c**w。存在一個唯一的「零向量」滿足 **0** + **v** = **v**。這三個條件列於問題的開頭。

這些條件要求任何向量空間。存在向量空間，它們不是列向量空間，並且向量空間除了 **R**ⁿ 之外，所有向量空間都必須遵守八條基本規則。

一個實數向量空間是一組「向量」以及加總和乘法規則。加總和乘法必須產生向量，這些向量在空間中。必須滿足這八條條件（通常沒有問題）。以下是三個除了 **R**ⁿ 之外的向量空間：

M  所有 2x2 的實數矩陣的向量空間。
F  所有實函數 f(x) 的向量空間。
Z  僅包含零向量的向量空間。

在 M 中，「向量」實際上是矩陣。在 F 中向量是函數。在 Z 中唯一的加總是 **0** + **0** = **0**。在這種情況下，我們可以加總：矩陣到矩陣，函數到函數，零向量到零向量。我們也可以將一個矩陣或一個函數乘以 4 或零向量乘以 4。結果仍然在 M 或 F 或 Z 中。這八個條件都已檢查。

函數空間 F 是無限維度的。一個更小的函數空間是 P₀ 或 P₁, 包含所有多項式 a₀ + a₁x + ... + aₙxⁿ 的度數 n。

空間 Z 是零維度的（由合理定義的維度）。它是最小的可能向量空間。我們猶豫不決地稱它為 **R**⁰，它沒有任何分量。你可能想知道沒有任何向量。向量空間 Z 包含一個零向量（零）。

你不會發現任何向量在 Z 之外。每個空間都具有其自身的定義和規則，但所有空間都遵守這八條基本規則。

**子空間**

在一個向量空間 V 中，如果一個集合 S 滿足以下條件，則 S 是 V 的子空間：

1.  **0** 位於 S 中。
2.  對於 S 中的任何向量 **v** 和 **w**，**v** + **w** 位於 S 中。
3.  對於 S 中的任何向量 **v** 以及任何純量 c，c**v** 位於 S 中。

如果這些條件成立，則 S 是 V 的子空間。例如，**R**² 中的所有向量 (**x**, 0) 形成一個子空間。它包含零向量，並且加總和純量乘法會產生向量 (**x**, 0)。

**R**² 中的直線通過原點也是一個子空間。它包含零向量，並且加總和純量乘法會產生直線上的向量。

**R**² 中的任何點都形成一個子空間（僅包含零向量）。

**R**² 中的整個平面 **R**² 也是一個子空間。
