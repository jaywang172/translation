## 128 章 3. 向量空間與子空間

**3** (a) 如果我們只保留正實數 $x > 0$ 在 $\mathbb{R}^1$ 中，哪些規則會被破壞？半直線不被允許。

(b) 將正實數與 $x + y$ 和 $cx$ 定義為通常的 $xy$ 和 $x^c$ 滿足八條規則。測試規則 7，當 $c = 3$, $x = 2$, $y = 1$ 時。( (然後 $x + y = 2$ 且 $cx = 8$)) 哪個數字被當作「零向量」？

**4** 矩陣 $A = \begin{bmatrix} 1 & 2 \\ 3 & 4 \end{bmatrix}$ 是所有 2x2 矩陣的向量空間 $M$ 中的一個「向量」。寫下這個空間中的零向量，向量 $\frac{1}{2}A$ 和向量 $-A$ 。哪個矩陣是在包含 $A$ 的最小子空間中？

**5** (a) 描述一個子空間 $M$ 包含 $A = \begin{bmatrix} 1 & 0 \\ 0 & 1 \end{bmatrix}$ 但不包含 $B = \begin{bmatrix} 0 & -1 \\ -1 & 0 \end{bmatrix}$。

(b) 如果一個 $M$ 的子空間包含 $A$ 和 $B$，它必須包含什麼？

(c) 描述一個 $M$ 的子空間包含非零對角矩陣。

**6** 函數 $f(x) = x^2$ 和 $g(x) = 5x$ 是在區間 $(-\infty < x < \infty)$ 上定義的 $F$ 的「向量」。組合 $3f(x) - 4g(x)$ 是函數 $h(x) = $ ______。

**7** 如果 $f(x)$ 和 $g(x)$ 的加法由 $f(x) + g(x)$ 產生函數 $f(cx)$，哪個規則被破壞？

**8** 如果「向量」 $f(x)$ 和 $g(x)$ 的和被定義為函數 $f(g(x))$，則函數 $f(g(x))$ 是「零向量」，$g(x) = x$。保持標量乘法 $c f(x)$ 不變，找出兩個被破壞的規則。

**問題 9-18 關於「子空間要求」：$x + y$ 和 $cx$ (然後所有線性組合 $cx + dy$ 都在子空間中)。**

**9** 一個要求可以被滿足，而另一個失敗。通過找到 $x$ 和 $y$ 的例子來證明。

(a) 一組向量在 $\mathbb{R}^2$ 中，對於其中 $x + y$ 都在集合中的向量 $x$ 和 $y$。

(b) 一組向量在 $\mathbb{R}^2$ 中，對於其中 $x + y$ 都在集合中的向量 $x$ 和 $y$。

**10** 哪些下列集合可能是 $\mathbb{R}^3$ 的兩個實際子空間？

(a) 平面的向量 $(b_1, b_2, b_3)$，其中 $b_1 = b_2$。

(b) $\mathbb{R}^3$ 中的所有向量 $(x, y, z)$，其中 $x = y = z$。

(c) 所有線性組合形式的向量 $c(1, 0, 0)$ 和 $d(0, 0, 2)$。

(d) $\mathbb{R}^3$ 中的所有向量 $(x, y, z)$，其中 $x + y + z = 0$。

(e) 描述所有向量形式的向量 $b_1 + b_2 + b_3$。

(f) 描述所有向量形式的向量 $c(1, 0, 0) + d(0, 1, 1)$，其中 $c = d$。
