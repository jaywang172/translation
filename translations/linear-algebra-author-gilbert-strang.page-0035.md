## 32 章 2. 解線性方程式

兩條線在點 x = 3, y = 1 處相交。該點同時滿足兩個方程式，因此是我們系統方程式的解。

**列 (ROWS)** 列圖顯示兩條線在一個單一點相交（解）。

現在轉向列圖。我想認識到相同的線性系統作為一個“向量方程式”。與其使用數字，我們需要使用向量。如果你將原始系統分解成列而不是行，你就會得到一個向量方程式：

**組合等於 b**
x [ 1 3 ] + y [ -2 2 ] = [ 1 1 ] = b. (2)

這兩個列向量在左側。問題是找到這些向量的組合，使其等於右側的向量 b。我們將第一個向量乘以 x，第二個向量乘以 y，然後相加。有了正確的選擇 x = 3 和 y = 1（與之前相同），這會產生 3(column 1) + 1(column 2) = b。

**欄 (COLUMNS)** 欄圖將左側的欄向量組合起來，產生右側的向量 b。

![Column picture: A combination of columns produces the right side (1,1).](https://i.imgur.com/a/9zX6J.png)

3(column 1) + 1(column 2) = b

![column 2](https://i.imgur.com/a/9zX6J.png) [ 1/3 column 1](https://i.imgur.com/a/9zX6J.png)

圖 2.2：欄圖：欄的組合產生右側 (1,1)。

圖 2.2 是兩個未知數的兩個方程式的“欄圖”。第一部分顯示兩個獨立的欄，第一個欄乘以 3。這種乘以一個純量（數字）是線性代數中的兩個基本運算之一：

**純量乘法**
3 [ 1 3 ] = [ 3 9 ].
