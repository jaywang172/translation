## 2.2. 消去法的概念

**步驟 3** 從第 3 列減去第 2 列的 1/1 倍。因此 4z = 8。

原始的 Ax = b 已經轉換成一個上三角矩陣 Ux = c：

```
2x + 4y - 2z = 2    Ax = b       2x + 4y - 2z = 2
4x + 9y - 3z = 8   has become    4y + 1z = 4
-2x - 3y + 7z = 10   Ux = c       4z = 8
```

(2)

目標已達成——正向消去從 A 到 U 完成。注意樞軸 2, 1, 4 沿著 U 的對角線。樞軸 1 和 4 存在於原始系統中。消去法將它們帶到一起。Ux = c 已經準備好進行反向代入，這非常快速：

(4z = 8 給 z = 2) (y + z = 4 給 y = 2) (equation 1 給 x = -1)

解為 (x, y, z) = (-1, 2, 2)。這三架飛機的行圖描述了這三個方程式。原始飛機是傾斜的，但最後一架飛機 z = 8 在消去後是水平的。

該列圖顯示一個組合的 4x 的列向量產生右側。這些係數以這種組合產生 -1, 2, 2 (解)：

```
Ax = (-1) [2] + 4 [4] + 2 [9] + (-2) [3] = [8]
           -2   -3   -7   10
```

(3)

數字 x, y, z 分別乘以列 1, 2, 3 在 Ax = b 中，並且在三角矩陣 Ux = c 中。對於 4 x 4 的問題，或者對於 n x n 的問題，消去法遵循相同的模式。這是整個過程，列從 A 到 U，列消去成功。

**列 1.** 使用第一列方程式來建立第一列下方的零。

**列 2.** 使用新的方程式 2 來建立第二列下方的零。

**列 3.** 在保持列 2 不變的情況下，找到所有 n 樞軸，並保持在樞軸上。

在第三列之後，我們有：

```
[ x x x x ]
[ 0 x x x ]  我們想要
[ 0 0 x x ]
[ 0 0 0 x ]
```

(4)

消去法的結果是形成一個上三角系統。如果它是無解的（never zero！），則 1, 2, 3, 4 中的某個列會導致變數的組合為零，但右側的常數不為零。如果有多個解，則某個列會全部為零，並且右側的常數也為零。

如果 A 是可逆的，則有一個獨一無二的解。在這種情況下，樞軸是所有對角線元素。

**例子** 考慮以下系統：

```
x + y + z = 3
2x + 3y + 3z = 9
x + 2y + 4z = 13
```

(5)

消去法會產生：

```
[ 1  1  1 ] [ x ] = [ 3 ]
[ 0  1  1 ] [ y ] = [ 3 ]
[ 0  0  1 ] [ z ] = [ 2 ]
```

(6)

反向代入給我們 z = 2, y = 1, x = 0。
