## 140 章 3. 向量空間與子空間

**3.2 B** 找出特殊解並描述 *Ax* = **0** 的完全解。

```
A₁ = [[0, 0, 0, 0],
      [0, 0, 0, 0]]

A₂ = [[3, 6],
      [1, 2]]

A₃ = [[1, 2],
      [2, 2]]
```

哪些是樞紐列？哪些是自由變數？*R* 在每種情況下是什麼？

**解** *A₁x* = **0** 有四個特殊解，它們是列 *s₁*, *s₂*, *s₃*, *s₄* 的 4 x 4 單位矩陣的所有解。零空間是 *R⁴* 的所有解。完全解是 *x* = *c₁s₁* + *c₂s₂* + *c₃s₃* + *c₄s₄* 在 *R⁴* 中。沒有樞紐列；所有變數都是自由的；簡約階梯形矩陣與 *A₁* 相同。

*A₂x* = **0** 只有一個特殊解 *s* = (-2, 1)。倍數 *x* = *cs* 給出完全解。第一列 *A₂* 是樞紐列，而 *x₂* 是自由變數。
行簡約矩陣 *R₂* 對 *A₂* 和 *R₃* 對 *A₃* = [[1, 2], [2, 2]] 有 1 在樞紐上：

```
A₂ = [[3, 6],
      [1, 2]] → R₂ = [[1, 2],
                      [0, 0]]

A₃ = [[1, 2],
      [2, 2]] → R₃ = [[1, 2],
                      [0, 0]]
```

請注意，*R₃* 只有一個樞紐列（第一列）。所有變數 *x₂*, *x₃*, *x₄* 都是自由的。有三個特殊解到 *A₃x* = **0** (且 *A₃* 的行數為 0)：

*s₁* = (-2, 1, 0, 0) *s₂* = (-1, 0, 1, 0) *s₃* = (-2, 0, 0, 1) 完全解 *x* = *c₁s₁* + *c₂s₂* + *c₃s₃*。
有了樞紐，*A* 有 *n* - *r* 個自由變數。*Ax* = **0** 有 *n* - *r* 個特殊解。
