## 3.5. Independence, Basis and Dimension 171

那種由零向量組成的平凡組合會讓每個作者頭痛。如果一個向量是其他向量的線性組合，那麼那個向量的係數就是 x = 1。

重點是，我們的定義不會挑出任何一個有罪的向量。A 的所有列都是相同的。我們檢查 Ax = 0，並且找到一個非零解或一個無解。最終，最好問最後一個（或第一個，或中間的一個）列是否是其他列的線性組合。

### Vectors that Span a Subspace

本書中的第一個子空間是由列空間填滿的。從向量 v₁,..., vₙ 開始，子空間由線性組合 x₁v₁ + ... + xₙvₙ 構成。列空間由這些列的線性組合構成。我們現在引入單詞“span”來描述這一點：列空間由列向量張成。

**定義** 一個集合的張成空間是其線性組合填滿的空間。

矩陣的列構成其列空間。它們可能是線性相依的。

**範例 2**  v₁ = [1, 0]ᵀ 和 v₂ = [0, 1]ᵀ 張成二維空間 R²。

**範例 3**  v₁ = [1, 0]ᵀ, v₂ = [1, 1]ᵀ, v₃ = [4, 7]ᵀ 也張成整個空間 R²。

**範例 4**  w₁ = [1, 1]ᵀ 和 w₂ = [-1, 1]ᵀ 僅張成 R² 中的一條線。w₁ 本身也是如此。

想想從 (0, 0) 出發的兩個向量在三維空間中張成一個平面。你的思維在通過線性組合填滿該平面。從數學上講，你還知道其他可能性：兩個向量可以張成 R³ 中的一條線，三個向量可以張成 R³ 中的一條線、一個平面或整個空間。

列向量張成一個列空間。這是一個新的子空間——由行向量張成。行向量的線性組合產生“行空間”。

**定義** 矩陣 A 的行空間是 Rⁿ 張成的由 A 的行向量張成的子空間。

行空間 A 是 C(Aᵀ)。它是 Aᵀ 的列空間。

矩陣 A 的行數為 m，列數為 n。它們是 Rⁿ 中的向量——或者，如果需要，它們可以被寫成列向量。那裡有一個快速的方法來解決這個問題：轉置矩陣 A。轉置後的 A 的列向量是 A 的行向量。所以，行空間 C(Aᵀ) 是 Aᵀ 的列空間。
