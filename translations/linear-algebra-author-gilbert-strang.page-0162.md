## 3.4. 完整解 $Ax = b$

159

**圖 3.3：完整解 = 一個特定解 + 所有零空間解。**

（圖示：一條線代表 $Ax = 0$ 的解，另一條線代表 $Ax = b$ 的解，兩線平行，且 $Ax = b$ 的線經過點 $b$；一個向量 $x_p$ 指向 $Ax = b$ 線上的某一點，一個向量 $x_n$ 指向 $Ax = 0$ 線上的某一點，且 $x = x_p + x_n$）

每個具有滿秩 ($r = min(m, n)$) 的矩陣 $A$ 都有以下特性：

1.  所有列向量都有領先元素，且 $R$ 沒有零列。
2.  $Ax = b$ 對於每個右側向量 $b$ 都有解。
3.  列空間是整個空間 $R^m$。
4.  存在 $n - r = n - m$ 個自由變數，在 $A$ 的零空間中。

在這一部分，我們有 $m$ 個領先變數，因此 $A^T$ 的列向量是“線性獨立的”。我們比定義線性獨立性更進一步。我們將總結四種可能性——這取決於秩 $r$。請注意 $r$、$m$、$n$ 是關鍵數字。

線性方程組的可能性取決於秩 $r$：

| $r = m$  and  $r = n$ | 平方且可逆 | $Ax = b$  has 1 solution |
| :------------------- | :--------- | :----------------------- |
| $r = m$  and  $r < n$ | 短而寬     | $Ax = b$  has $\infty$ solutions |
| $r < m$  and  $r = n$ | 瘦而高     | $Ax = b$  has 0 or 1 solution |
| $r < m$  and  $r < n$ | 不滿秩     | $Ax = b$  has 0 or $\infty$ solutions |

簡約行階梯形式 $R$ 屬於同一類別，如矩陣 4。在領先列出現的情況下，我們可以顯示這些可能性對於 $R$。對於 $Rx = d$ (和原始 $Ax = b$) 來說， $d$ 必須在 $m - r$ 個零向量中。

| 四種類型 | $R = [I]$ | $[I \ F]$ | $[0 \ I]$ | $[0 \ 0]$ |
| :------- | :-------- | :-------- | :-------- | :-------- |
| 秩       | $r = m = n$ | $r = m < n$ | $r = n < m$ | $r < m, r < n$ |

案例 1 和 2 具有滿秩 $r = m$。案例 1 和 3 具有滿列秩 $r = n$。案例 4 是最一般的理論和實踐。

請注意，我的類別基於從 $x_p$ 到 $x_n$ 的反向追蹤 $R$ 。相反，從閱讀完整解直接從 $Rx = d$ 得到，我們發現 $x = c$。
