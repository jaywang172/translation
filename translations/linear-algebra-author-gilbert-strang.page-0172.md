## 3.5. 獨立性、基底與維度 169

**圖 3.4：** 獨立向量 v₁, v₂, v₃。只有 0v₁ + 0v₂ + 0v₃ 給出向量 0。
相依向量 w₁, w₂, w₃。組合 w₁ - w₂ + w₃ = (0,0,0)。

**定義** 向量序列 v₁, ..., vn 被稱為線性獨立，如果只有組合 v₁x₁ + v₂x₂ + ... + vₙxₙ = 0 才會發生當所有 x 都為零時。

(1)
x₁v₁ + x₂v₂ + ... + xₙvₙ = 0  僅在所有 x 都為零時發生。

如果一個組合給出 0，而 x 值並非全為零，則這些向量是相依的。
正確的說法：“向量序列是線性獨立的。” 可接受的簡寫：“向量是獨立的。” 不可接受的：“矩陣是獨立的。”
一個向量序列可以是獨立的或相依的。它們可以被組合以給出零向量（以非零 x 值）嗎？ 這是關鍵問題：哪些向量的組合可以給出零向量？ 我們從 R² 中的一些小例子開始。

(a) 向量 (1,0) 和 (0,1) 是獨立的。

(b) 向量 (1,0) 和 (1,0.00001) 是獨立的。

(c) 向量 (1,1) 和 (-1,-1) 是相依的。

(d) 向量 (1,1) 和 (0,0) 是相依的，因為存在零向量。

(e) 在 R² 中，任何三個向量 (a,b) 和 (c,d) 和 (e,f) 是相依的。

幾何上，(1,1) 和 (-1,-1) 位於通過原點的一條線上。它們是相依的。
為了使用定義，找到 x₁ 和 x₂，使得 x₁(1,1) + x₂(-1,-1) = (0,0)。
這與求解以下方程組相同：

[ 1  -1 ] [ x₁ ] = [ 0 ]
[ 1  -1 ] [ x₂ ] = [ 0 ]

對於 x₁ = 1 且 x₂ = 1。

向量的列是相依的，如果存在非零向量在零空間中。為什麼不是？
