## 2.4 矩陣運算規則

我將從基本事實開始：矩陣是一個由數字或「元素」組成的矩形陣列，它有 *m* 列和 *n* 行，稱為 *m* × *n* 矩陣。如果形狀相同，矩陣可以相加，也可以乘以任何常數。以下是一些矩陣 *A* + *B* 和 2*A* 的例子，對於 3 × 2 矩陣：

```
[ 1  2 ]   [ 2  2 ]   [ 3  4 ]   [ 1  2 ]   [ 2  4 ]
[ 3  4 ] + [ 4  4 ] = [ 7  8 ]   2 [ 2  3 ] = [ 6  8 ]
[ 0  9 ]   [ 9  9 ]   [ 0  0 ]   [ 0  0 ]   [ 0  0 ]
```

矩陣的加法就像向量一樣，一次加一個元素。我們可以將列向量視為只有一列（因此 *n* = 1）的矩陣。矩陣 -*A* 源自乘以 -1（反轉所有元素）。將 *A* 加到 -*A* 產生零矩陣，其中所有元素都為零。這一切都符合常理。

矩陣中第 *i* 列和第 *j* 行的元素稱為 *a<sub>ij</sub>* 或 *A(i, j)*。第一行的元素是 *a<sub>11</sub>*, *a<sub>12</sub>*, ..., *a<sub>1n</sub>*。最後一個元素是 *a<sub>m1</sub>*，從上到下，從左到右。

矩陣加法很簡單，真正的問題是矩陣乘法。我們如何將 *A* 和 *B* 相乘？當 *A* 是 3 × 2 時，*B* 可以是 2 × 1（向量）或 2 × 2（平方）或 2 × 20。每一列 *B* 都必須乘以 *A*。我將以矩陣乘法的零產品規則開始，然後返回到這個主題：*A* 乘以 *B* 等於 *A* 的列乘以 *B* 的行。一個挑戰問題將證明這一點。

假設 *A* 是 *m* × *n* 且 *B* 是 *n* × *p*。我們可以計算 *AB* 的結果：*AB* 是 *m* × *p*。

```
(m × n) × (n × p) = (m × p)   [ n columns ] [ p columns ] [ p columns ]
```

矩陣的每一列都包含 *m* 個元素。*A* 有 *n* 列，*B* 有 *p* 列。為了使 *AB* 有意義，*A* 的列數必須等於 *B* 的行數。因此，如果 *A* 是 2 × 3 且 *B* 是 3 × 1，則 *AB* 是 2 × 1。如果 *A* 是 3 × 2 且 *B* 是 1 × 3，則 *AB* 是 3 × 3。

**為什麼這樣做？**

在工程學中，矩陣乘法代表線性變換。假設 *A* 是 2 × 2 矩陣，代表旋轉。然後 *AB* 代表先進行 *B* 變換，然後進行 *A* 變換。矩陣乘法將這些變換組合在一起。

**一個挑戰問題：** 證明如果 *A* 是 *m* × *n* 矩陣且 *B* 是 *n* × *p* 矩陣，則 (*AB*) *C* = *A* (*BC*)。這稱為結合律。
