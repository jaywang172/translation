4.1. 四個子空間的正交性 205

問題 24–30 關於相互垂直的行與列。

24 假設一個 $n \times n$ 矩陣是可逆的：$AA^{-1} = I$。那麼 $A^{-1}$ 的第一行與 $A$ 的哪些列所生成的空間正交？

25 如果 $A$ 的各行都是單位向量且彼此相互垂直，請找出 $A^T A$。

26 構建一個 $3 \times 3$ 且沒有零項的矩陣 $A$，使其各行彼此相互垂直。計算 $A^T A$。為什麼它是一個對角矩陣？

27 直線 $3x + y = b_1$ 與 $6x + 2y = b_2$ 是 \_\_\_\_\_。如果 \_\_\_\_\_，則它們是同一條直線。在這種情況下，$(b_1, b_2)$ 與向量 \_\_\_\_\_ 垂直。矩陣的零空間是直線 $3x + y = \_\_\_\_\_$。該零空間中的一個特解向量是 \_\_\_\_\_。

28 為什麼以下每個陳述都是錯誤的？

(a) $(1, 1, 1)$ 與 $(1, 1, -2)$ 垂直，因此平面 $x + y + z = 0$ 與 $x + y - 2z = 0$ 是正交子空間。

(b) 由 $(1, 1, 0, 0, 0)$ 與 $(0, 0, 0, 1, 1)$ 生成的子空間，是由 $(1, -1, 0, 0, 0)$ 與 $(2, -2, 3, 4, -4)$ 生成之子空間的正交補空間。

(c) 僅在零向量處相交的兩個子空間是正交的。

29 找出一個矩陣，使得 $v = (1, 2, 3)$ 同時位於列空間與行空間中。找出另一個矩陣，使得 $v$ 同時位於零空間與行空間中。$v$ 不能同時存在於哪些子空間對中？
