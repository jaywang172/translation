## 34 章：解線性方程組

1. 消去法透過一系列矩陣步驟 $E_{ij}$ 將矩陣 $A$ 轉換為三角矩陣 $U$。

2. 逆矩陣 $A^{-1}$ 以相反的順序將 $U$ 轉換回原始矩陣 $A$。

3. 在矩陣語言中，矩陣分解 $A = LU = (\text{下三角矩陣})(\text{上三角矩陣})$。

4. 消去法成功的前提是 $A$ 可逆。 (它可能需要交換行。)

最常用的演算法在計算機科學中需要更多步驟 (MATLAB 稱之為 `lu`)。線性代數超越了方形可逆矩陣的範圍。對於 $m$ 行 $n$ 列的矩陣， $Ax = b$ 可能有多個解。這些解會落在一個向量空間中。矩陣 $A$ 的秩決定了這個向量空間的維度。

所有關於維度的內容在第 3 章中，我不想在那裡停留。

### 三個未知數的三個方程式

三個未知數為 $x, y, z$。我們有三個線性方程式：

$Ax = b$

$\begin{cases}
x + 2y + 3z = 6 \\
2x + 5y + 2z = 4 \\
6x - 3y + z = 2
\end{cases}$  (3)

我們尋找數字 $x, y, z$，它們可以同時滿足這三個方程式。這些數字可能存在，也可能不存在。對於這個系統，我們通常期望有一個解。在求解問題之前，我們以兩種方式視覺化它：

**ROW** 行圖像：三條平面在一個點相交。

**COLUMN** 列圖像：三列組合產生向量 $(6, 4, 2)$。

在行圖像中，每個方程式產生一個平面在三維空間中。第一個平面，來自方程式 $x + 2y + 3z = 6$。這個平面穿過 $x$ 軸、 $y$ 軸和 $z$ 軸在點 $(6, 0, 0)$、 $(0, 3, 0)$ 和 $(0, 0, 2)$。這三個點定義了這個平面的範圍。

向量 $(x, y, z) = (0, 0, 0)$ 並不能滿足 $x + 2y + 3z = 6$。因此，這個平面不包含原點。平面 $x + 2y + 3z = 6$ 與平面 $x + 2y + 3z = 0$ 平行，而後者穿過原點，並沿著 $x$ 軸和 $y$ 軸移動。

第二個平面來自方程式 $2x + 5y + 2z = 4$。這個平面穿過 $x$ 軸、 $y$ 軸和 $z$ 軸在點 $(2, 0, 0)$、 $(0, \frac{4}{5}, 0)$ 和 $(0, 0, 2)$。

第三個平面來自方程式 $6x - 3y + z = 2$。這個平面穿過 $x$ 軸、 $y$ 軸和 $z$ 軸在點 $(\frac{1}{3}, 0, 0)$、 $(0, -2, 0)$ 和 $(0, 0, 2)$。

這三個平面相交於一點。我們可以透過求解線性方程式來找到這個點 $(1, 1, 1)$。
