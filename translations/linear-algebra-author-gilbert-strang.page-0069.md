**31** 找到消去矩陣 E<sub>21</sub>、E<sub>32</sub> 和 E<sub>43</sub>，將 K 變換為 U：

```
K = [ 2 -1  0  0 ]
    [ 0  2 -1  0 ]
    [ 0  0  2 -1 ]
    [ 0  0 -1  2 ]
```

```
U = [ 2 -1  0  0 ]
    [ 0 3/2 -1  0 ]
    [ 0  0 4/3 -1 ]
    [ 0  0  0 5/4 ]
```

應用這三個步驟到單位矩陣 I，以得到 E<sub>43</sub>E<sub>32</sub>E<sub>21</sub>。
