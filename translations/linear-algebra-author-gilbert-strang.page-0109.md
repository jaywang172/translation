**24** 哪些可逆矩陣允許 `A = LU`（消去，沒有列交換）？
好問題！觀察 `A` 的上三角子矩陣。

所有上三角 `k` 乘以 `k` 子矩陣 `Ak` 必須是可逆的（大小 `k = 1, ..., n`）。

解釋這個答案：`Ak` 分解成

因為 `LU = [ Lk  0 ] [ Uk * ]`
            `[ *   0 ] [ *   * ]`

**25** 對於 6x6 的第二差分常數對角矩陣 `K`，求出樞軸和乘數，使得 `K = LU`。（`L` 和 `U` 將只有兩個非零對角線，因為 `K` 有三條。）找出 `L⁻¹` 的 `(i, j)` 元素，或者使用像 MATLAB 這樣的軟體使用 `inv(L)` 或尋找一個漂亮的模式。

**26** 如果你印出 `K⁻¹`，它看起來不太好。但如果你印出 `7K⁻¹`（當 `K` 是 6x6 時），它看起來會很棒。寫出 `7K⁻¹` 的形式，遵循這個模式：

1.  第一列和第一行是 (6, 5, 4, 3, 2, 1)。
2.  在主對角線之上，對角線 `i` 是 `i` 倍的行 1。
3.  在主對角線之下，列 `j` 是 `j` 倍的列 1。

將 `K` 乘以 `7K⁻¹` 來產生 `7I`。這裡對於 `n = 3` 的模式：

3 by 3 case

```
(7)(K⁻¹) = [ 2 -1  0 ] [ 3  2  1 ]
           [-1  2 -1 ] [ 2  4  2 ]
           [ 0 -1  2 ] [ 1  2  4 ]
```

The determinant of `K` is 4.
