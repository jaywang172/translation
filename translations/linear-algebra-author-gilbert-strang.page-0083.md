## 挑戰問題

**36** 實用問題 假設 $A$ 是 $m$ 行 $n$ 列， $B$ 是 $n$ 行 $p$ 列，且 $C$ 是 $p$ 行 $q$ 列。那麼 $(AB)C$ 的乘法次數是 $mnp + mpq$。相同的答案來自 $A(BC)$，但需要 $mnq + npq$ 個獨立的乘法。注意 $npq$ 個答案。

(a) 如果 $A$ 是 $2$ 行 $4$ 列， $B$ 是 $4$ 行 $7$ 列，且 $C$ 是 $7$ 行 $10$ 列，你更喜歡 $(AB)C$ 還是 $A(BC)$？
(b) 對於 $N$-組件向量，你會選擇 $(u^T v)w^T$ 還是 $u^T (v w^T)$？
(c) 將 $mnpq$ 除以 $mnpq$ 以證明 $(AB)C$ 比 $A(BC)$ 更快，當 $n^{-1} + q^{-1} < m^{-1} + p^{-1}$。

**37** 要證明 $(AB)C = A(BC)$，使用 $B$ 的列向量 $b_1, \dots, b_n$。首先證明 $C$ 只有一個條目 $c_{1j}$。

$AB$ 具有列 $Ab_1, \dots, Ab_n$，然後 $(AB)C$ 等於 $c_{1j}Ab_1 + \dots + c_{nj}Ab_n$。
$Bc$ 具有列 $b_1c_1 + \dots + b_nc_n$，然後 $A(BC)$ 等於 $A(b_1c_1 + \dots + c_nb_n)$。
線性性質給予等式 $c_{1j}Ab_1 + \dots + c_{nj}Ab_n$。這證明 $(AB)C = A(BC)$。這個結果對 $C$ 的所有其他列都成立。應用到逆矩陣：
如果 $BA = I$ 且 $AC = I$，證明 $A$ 是 $B$ 的右逆矩陣。
