## 3.5. Independence, Basis and Dimension 175

向量空間 $\mathbb{R}^n$ 的維度是我們所知的。現在我們引入重要的詞彙**維度**，用於其他向量空間。

**定義** 一個空間的**維度**是每個基底所包含的向量數量。

這符合我們的直覺。穿過 $\mathbf{v} = (1.5, 2)$ 的線的維度為 1。它是與向量 $\mathbf{v}$ 垂直的直線的子空間 $x + 5y + 2z = 0$。這個平面有維度 2。為了證明它，我們找到一個基底 $(-5, 1, 0)$ 和 $(-2, 0, 1)$。這個維度是 2，因為基底包含兩個向量。

這個平面是矩陣 $A = \begin{bmatrix} 1 & 5 & 2 \end{bmatrix}$ 的零空間，它有兩個自由變數。我們的基底向量 $(-5, 1, 0)$ 和 $(-2, 0, 1)$ 是 $Ax = 0$ 的“特殊解”。下一個部分顯示 $n - r$ 個特殊解總是給出零空間的一個基底。$C(A)$ 的維度是 $r$，而零空間 $N(A)$ 的維度是 $n - r$。

請注意線性代數的語言。我們從不說一個“空間的秩”或“基底的維度”。這些術語沒有意義。它指的是矩陣的列空間的維度，它等於矩陣的秩。

## Bases for Matrix Spaces and Function Spaces

“獨立性”、“基底”和“維度”這些詞語與列向量相關。我們可以問三個矩陣 $A_1, A_2, A_3$ 是否獨立。當它們在所有 3 x 3 矩陣中，一些組合可能會給出零矩陣。我們可以問這個 3 x 3 矩陣空間的維度是多少？(它是 12)。

在微分方程中，$d^2y/dx^2 = y$ 有一個解的空間。一個基底是 $y = e^x$ 和 $y = e^{-x}$。計算這兩個函數給出了維度 2 的解空間。(維度是 2，因為二階導數)。

矩陣空間和函數空間看起來可能有點奇怪，直到你習慣了它們。但它們具有向量，你已經了解了基底和維度的概念。

**矩陣空間** 向量空間 $M$ 包含所有 2 x 2 矩陣。它的維度是 4。

例如，$\begin{bmatrix} 1 & 0 \\ 0 & 0 \end{bmatrix}$, $\begin{bmatrix} 0 & 1 \\ 0 & 0 \end{bmatrix}$, $\begin{bmatrix} 0 & 0 \\ 1 & 0 \end{bmatrix}$, $\begin{bmatrix} 0 & 0 \\ 0 & 1 \end{bmatrix}$ 形成一個基底。這些矩陣是獨立的，因為沒有任何非零的 $c_1, c_2, c_3, c_4$ 使得

$c_1 \begin{bmatrix} 1 & 0 \\ 0 & 0 \end{bmatrix} + c_2 \begin{bmatrix} 0 & 1 \\ 0 & 0 \end{bmatrix} + c_3 \begin{bmatrix} 0 & 0 \\ 1 & 0 \end{bmatrix} + c_4 \begin{bmatrix} 0 & 0 \\ 0 & 1 \end{bmatrix} = \begin{bmatrix} 0 & 0 \\ 0 & 0 \end{bmatrix}$.

**函數空間** 函數空間包含所有 $x$ 的多項式，其最高次數小於或等於 $n$。它的維度是 $n+1$。

例如，對於 $n = 2$，基底是 $1, x, x^2$。對於任何多項式 $p(x) = a + bx + cx^2$，我們有 $p(x) = a(1) + b(x) + c(x^2)$。

**向量空間的維度**

*   $\mathbb{R}^n$ 的維度是 $n$。
*   $m \times n$ 矩陣的空間的維度是 $mn$。
*   所有多項式的空間，其最高次數小於或等於 $n$ 的維度是 $n+1$。
*   所有 $n$ 階連續函數的空間的維度是無窮大。
