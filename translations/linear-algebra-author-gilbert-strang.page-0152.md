## 3.3. 行秩與行簡化梯形式 149

**矩陣列式簡化碼**

沒有任何方法能比 `rref` 更能勝任列式簡化。`lu` 教學碼會再次被呼叫。當然 `rref(R)` 會再次呼叫 `R`。

MATLAB: `[R, pivcol] = rref(A)`  教學碼: `[E, R] = elim(A)`

額外的輸出 `pivcol` 給出了樞軸欄的數量。它們與在 `A` 和 `R` 中輸出的 `E` 中的樞軸欄相同。教學碼中的一個 `m` 消去矩陣，將原始 `A`（無論它原本是什麼）簡化為行簡化形式 `R`。

`EA = R`

矩陣 `E` 是基本矩陣 `Eij` 和對角矩陣 `D⁻¹` 的乘積。`Pij` 交換列。對角線 `D⁻¹` 將其樞軸列除以其樞軸以產生 1。

如果我們想要 `E`，我們可以將列式簡化應用到矩陣 `[A | I]`，其中 `I` 是 `n` 個欄的單位矩陣。乘積 `E` 的所有基本矩陣，將 `A` 簡化為 `R`，並將 `I` 簡化為 `E`。整個擴增矩陣被矩陣 `E` 乘以：

`E [A | I] = [R | E]`  (7)

這正是第 2 章中計算 `A⁻¹` 的“高斯-喬丹”方法。當 `A` 是方陣且可逆時，其**行簡化梯形式**是 `I`。然後 `EA` 變成 `EA = I`。在此可逆情況下，`E` 是 `A⁻¹`。本章將進一步說明 `E = A⁻¹`。

**關鍵概念回顧**

1.  矩陣 `A` 的秩是樞軸的數量（在 `rref(A)` 中是 1 的數量）。
2.  樞軸欄的 `A` 和 `R` 都在相同的列中 `pivcol`。
3.  這些樞軸欄是較早欄的組合。
4.  `n - r` 個自由欄是較早欄（樞軸欄）的組合。
5.  這些組合（使用 `F` 從 `R` 取得）給出了 `n - r` 個 `Ax = 0` 和 `Rx = 0` 的特殊解。它們是零空間矩陣 `N` 的 `n - r` 個欄。
