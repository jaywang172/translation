## 114 章節 2. 解線性方程式

**工作範例**

**2.7 A** 應用置換矩陣 *P* 到矩陣 *A* 的列上，可以破壞其對稱性：

```
P = | 0  1  0 |
    | 0  0  1 |
    | 1  0  0 |
```

```
A = | 1  4  5 |
    | 4  2  6 |
    | 5  6  3 |
```

```
PA = | 4  2  6 |
     | 5  6  3 |
     | 1  4  5 |
```

什麼置換 *P* 應用到 *PA* 的列上，可以讓 *PA* 的主對角線上的數字為 1, 2, 3 (不一定按順序排列)？ 證明 *Q* 是 *P*<sup>T</sup>，所以 *PAQ = PAP*<sup>T</sup>。

**解答** 為了恢復對稱性並將 “2” 放在主對角線上，*PA* 的第二列必須移動到第一列。*PA* 的第三列 (包含 “3”) 必須移動到第二列。然後 “1” 移動到第三列的位置。這個矩陣 *Q* 允許的列置換是：

```
PA = | 4  2  6 |
     | 5  6  3 |
     | 1  4  5 |
```

```
Q = | 0  1  0 |
    | 0  0  1 |
    | 1  0  0 |
```

```
PAQ = | 2  6  4 |
      | 6  3  5 |
      | 4  5  1 |
```

是對稱的。

矩陣 *Q* 是 *P*<sup>T</sup>。這個選擇總是恢復對稱性，因為 *PAP*<sup>T</sup> 總是對稱的 (它的轉置是 (*PAP*<sup>T</sup>)<sup>T</sup> = *P* *A* *P*<sup>T</sup>)。矩陣 *Q* 也是 *P*<sup>-1</sup>，因為它是 *P* 的逆置換矩陣的轉置。

如果 *D* 是一個對角矩陣，我們正在尋找將 *PDP*<sup>T</sup> 變成對角矩陣。當 *P* 向下移動一行到第三行時，我們將第三列移動到第一列。 (1, 1) 的條目向下移動到 (3, 1) 並且移動到 (3, 3)。

**2.7 B** 找到矩陣 *A* 的對稱分解 *A = LDL*<sup>T</sup>。*A* 是可逆的嗎？ 另外找到 *PQ = LU* 分解的 *Q*，需要多少次交換？

**解答** 為了將 *A* 分解成 *LDL*<sup>T</sup>，我們消去主對角線以下的元素：

```
A = | 1  4  5 |
    | 4  2  6 |
    | 5  6  3 |
```

```
-> | 1  4  5 |
    | 0 -14 -14 |
    | 0 -14 -22 |
```

```
-> | 1  4  5 |
    | 0 -14 -14 |
    | 0  0  -8 | = U
```

乘數是 *l<sub>21</sub>* = 4 並且 *l<sub>31</sub>* = 5 並且 *l<sub>32</sub>* = 1。所以 *D* 的元素是 1, -14, -8。

對稱分解
形式化：

```
A = LDL^T = | 1  0  0 |   | 1  4  5 |   | 1  0  0 |
              | 0 -14 0 | * | 4  2  6 | * | 0 -1  0 |
              | 0  0 -8 |   | 5  6  3 |   | 0  0  1 |
```

這個矩陣 *A* 是可逆的，因為所有主對角線上的元素都不為零。*U* 是 *A* 的上三角矩陣。為了將 *U* 分解成 *LU*，我們需要 1 次交換。
