第 2 章：解線性方程組

**奇異性與可逆性**

我們回歸到中心問題：哪些矩陣具有反矩陣？本節開頭的命題是：一個 *n* × *n* 矩陣 *A* 恰好在它具有 *n* 個主元的情況下可逆。（允許行交換。）現在我們將通過高斯-約旦消元法來證明這一點：

1.  具有 *n* 個主元，消元法解方程組 *Ax* = *eᵢ*。這些列向量 *xᵢ* 構成 *A⁻¹*。因此 *A⁻¹* = *I* 且 *A⁻¹* 至少是一個右反矩陣。

2.  消元法實際上是一系列由 *E* 和 *P* 矩陣以及 *D⁻¹* 矩陣組成的乘法序列：

    ( *D⁻¹* … *E* … *P* … *E*) *A* = *I*。 (9)

    *D⁻¹* 除以主元。*E* 矩陣產生主元上方和下方的零。如果需要，*P* 將交換行（參見第 2.7 節）。因此，乘積矩陣在方程 (9) 中顯然是一個左反矩陣。具有 *n* 個主元的我們已經得到 *A⁻¹A* = *I*。

    右反矩陣等於本節開頭的注 2 中所說的左反矩陣。因此，一個具有完整主元集的方陣始終具有一個兩側反矩陣。

    反向推理現在將表明 *A* 必須具有 *n* 個主元才能使 *AC* = *I*。（然後我們將推斷 *C* 也是 *A* 的左反矩陣，並且 *CA* = *I*。）這裡有一條路線可以得出這些結論：

    1.  如果 *A* 沒有 *n* 個主元，消元法將導致一列零。
    2.  這些消元步驟可以被視為一個可逆矩陣 *M*。因此 *MA* 的某一行是零。
    3.  如果 *AC* = *I* 成立，那麼 *MAC* = *M*。零行乘以 *M*，得到 *M* 的零行。
    4.  一個可逆矩陣 *M* 可以有一列零嗎？*A* 必須具有 *n* 個主元才能使 *AC* = *I*。

    這個論證包含四個步驟，但結果是重要且值得注意的。

    消元法為方陣的可逆性提供了一個完整的測試。*A⁻¹* 存在（且高斯-約旦消元法找到它）恰好在 *A* 具有 *n* 個主元的情況下。上述論證表明更多：

    如果 *AC* = *I*，那麼 *CA* = *I* 且 *C* = *A⁻¹*

**例 5** 如果 *L* 是下三角矩陣，且對角線上為 1，那麼 *L* 是 *L⁻¹*。
