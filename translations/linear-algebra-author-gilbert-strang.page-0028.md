## 1.3. 矩陣 25

距離旅行的導數 (v) 是速度 (b)。距離 x(t) 的積分是從 x(0) = 0 開始的距離。以同樣的方式，差異從 x₀ = 0 開始。這個零點使模式完整，當我們寫 x₁ - x₀ 作為 Δx 的第一個分量時（我們剛剛寫了 x₁）。

請注意另一個類似於平方數 0, 1, 4, 9 的類比。x(t) = t² 的導數是 2t。一個完美的類比產生了偶數 2, 4, 6 在 t = 1, 2, 3 時。但差異並不像導數那樣，而矩陣 Δx 產生了 2t + 1 (這些被稱為“反向差異”)，它們以 t - ½ 為中心。

x(t) - x(t-1) = t² - (t-1)² = t² - (t² - 2t + 1) = 2t - 1. (9)

問題集將會展示“前向差異”產生 2t + 1。一個更好的選擇（並未在微積分課程中看到）是使用中心差異，它使用 x(t + 1) - x(t - 1)。將 x(t) = t² 除以距離 Δt 從 t - 1 到 t + 1，這恰好是 2：

中心差異的 x(t) = t²  =  (t + 1)² - (t - 1)² / 2 = 2t  精確地。 (10)

差異矩陣很棒。中心差異是最佳的。我們的第二個例子是不可逆的。

## 循環差異

這個例子保持相同的列 u 和 v，但將 w 更改為一個新的向量 w*：

第二個例子
u =  [1, 0]ᵀ  v =  [1, 1]ᵀ  w* = [-1, 0]ᵀ

現在線性組合 u, v, w* 導致一個循環差異矩陣 C：

循環
C =  [1, 0, -1; -1, 1, 0; 0, -1, 1]  [x₁, x₂ , x₃] = b. (11)

這個矩陣 C 是不可逆的。它不是簡單的求解 x，因為我們被給定了 b。實際上，目的是找到滿足 Cx = b 的 x，因為這些方程式沒有無限多個解。

C x = b 的唯一解是 x₁ = x₂ = x₃ = 0。
